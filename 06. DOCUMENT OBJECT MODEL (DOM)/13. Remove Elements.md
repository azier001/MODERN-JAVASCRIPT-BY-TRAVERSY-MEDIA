<a name="tips--best-practices"></a>
## ğŸ’¡ Tips & Best Practices

### âœ… DO (Lakukan)

   ```javascript

   element.remove();
   
   // Butuh control dari parent â†’ pakai removeChild()
   parent.removeChild(child);
   ```

   ```javascript

     if (element) {
       element.remove();
     } else {
       console.log('Element not found!');
     }
   }
   ```

3. **Beri nama fungsi yang jelas**
   ```javascript
   // âœ… Good
   function removeFirstItem() { }
   function removeItemByIndex(index) { }
   
   // âŒ Bad
   function remove1() { }
   function func() { }
   ```

---

### âŒ DON'T (Hindari)

1. **Jangan hapus element yang tidak ada**
   ```javascript
   // âŒ Bad - bisa error!
   document.querySelector('#notexist').remove();
   
   // âœ… Good - cek dulu
   const el = document.querySelector('#notexist');
   if (el) el.remove();
   ```

2. **Jangan hardcode index**
   ```javascript
   // âŒ Bad
   document.querySelectorAll('li')[2].remove();
   
   // âœ… Good - pakai parameter
   function removeItem(index) {
     document.querySelectorAll('li')[index].remove();
   }
   ```

3. **Jangan lupa index dimulai dari 0**
   ```javascript
   // âŒ Bad - akan hapus item ke-2!
   const items = document.querySelectorAll('li');
   items[1].remove(); // Orange Juice, bukan Apples
   
   // âœ… Good - kurangi 1 jika user input 1-based
   function removeItem(itemNumber) {
     const items = document.querySelectorAll('li');
     items[itemNumber - 1].remove();
   }
   ```
