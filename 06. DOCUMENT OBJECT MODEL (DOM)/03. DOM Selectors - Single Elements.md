<a name="mendapatkan-atribut"></a>
### ğŸ” Mendapatkan Atribut

Ada **2 cara** untuk mendapatkan atribut dari element:

#### **Cara 1: Menggunakan Property** (untuk atribut umum)


> ğŸ’¡ **Tips:** Gunakan `className` bukan `class` karena `class` adalah reserved word di JavaScript!

#### **Cara 2: Menggunakan getAttribute()** (untuk semua atribut)

```javascript

// Output: 'app-title'
```

**Kapan pakai yang mana?**
- âœ… Gunakan **property** untuk atribut umum (id, className)
- âœ… Gunakan **getAttribute()** untuk atribut custom atau kurang umum

## ğŸš€ querySelector() - Selector Modern

### ğŸ’¡ Kenapa querySelector() Game Changer?

Sebelum `querySelector()` ada, kita **harus** menggunakan ID:
```javascript

```

Sekarang dengan `querySelector()`, kita bisa pilih **APAPUN** seperti di CSS!
```javascript

document.querySelector('#app-title');   // âœ… By ID
document.querySelector('.container');   // âœ… By class
document.querySelector('li:first-child'); // âœ… Pseudo selector
```

> ğŸ‰ **Fun Fact:** `querySelector()` datang dengan update ES6/ES2015 dan membuat jQuery jadi kurang relevan untuk DOM manipulation!

### 1ï¸âƒ£ Selector by Tag

Memilih element berdasarkan **nama tag HTML**.

Kamu tidak harus selalu menggunakan `document.querySelector()`! Kamu bisa gunakan `querySelector()` pada **element lain** juga!

### ğŸ’¡ Konsep

```javascript

```

```javascript

// Sekarang 'Apples' berwarna biru!
```
