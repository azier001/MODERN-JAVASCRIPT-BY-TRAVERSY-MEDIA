<a name="method-utama-yang-digunakan"></a>
## ğŸ› ï¸ Method Utama yang Digunakan

<a name="createelement"></a>
### 1. `createElement()`

Membuat elemen HTML baru dalam bentuk **node** (bukan string!).

```javascript

const button = document.createElement('button'); // Membuat element <button>
```

> âš ï¸ **Penting**: Hasil dari `createElement()` adalah **node/object**, bukan string HTML!

---

<a name="appendchild"></a>
### 2. `appendChild()`

Menambahkan elemen sebagai **child** (anak) dari elemen lain.

```javascript
document.querySelector('.items').appendChild(li); // Menambahkan li ke dalam ul.items
```

---

<a name="createtextnode"></a>
### 3. `createTextNode()`

Membuat node teks (bukan elemen HTML).

```javascript
const text = document.createTextNode('Eggs'); // Membuat teks "Eggs"
li.appendChild(text); // Menambahkan teks ke dalam li
```

## ğŸš€ Metode 1: Quick & Dirty (innerHTML)

Metode ini lebih cepat untuk ditulis tapi **kurang efisien** untuk performa.

<a name="kode-lengkap-metode-1"></a>
### ğŸ“ Kode Lengkap Metode 1

<a name="penjelasan-step-by-step-metode-1"></a>
### ğŸ” Penjelasan Step-by-Step Metode 1

#### **Step 1: Membuat Element `<li>`**
```javascript
const li = document.createElement('li');
```
- Membuat elemen list item baru
- Saat ini masih **kosong** dan belum ada di DOM

---

#### **Step 2: Mengisi Konten dengan `innerHTML`**
```javascript
li.innerHTML = `${item}
  <button class="remove-item btn-link text-red">
    <i class="fa-solid fa-xmark"></i>
  </button>`;
```
- Menggunakan **template literals** (backticks) untuk menulis HTML
- `${item}` akan diganti dengan parameter yang dimasukkan (contoh: "Eggs")
- Langsung menulis HTML sebagai string

> ğŸ’¡ **Template Literals**: Menggunakan backticks (\` \`) agar bisa memasukkan variabel dengan `${}`

---

#### **Step 3: Menambahkan ke DOM**
```javascript
document.querySelector('.items').appendChild(li);
```
- `document.querySelector('.items')` = mencari element dengan class "items" (yaitu `<ul>`)
- `appendChild(li)` = menambahkan `<li>` yang sudah dibuat ke dalam `<ul>`

---

<a name="kelebihan--kekurangan-metode-1"></a>
### âš–ï¸ Kelebihan & Kekurangan Metode 1

#### âœ… **Kelebihan:**
- Kode lebih **singkat** dan mudah dibaca
- Lebih **cepat** untuk ditulis
- Cocok untuk konten HTML yang kompleks

#### âŒ **Kekurangan:**
- **Kurang efisien**: Browser harus re-parse dan recreate semua DOM nodes di dalam `<ul>`
- **Event handlers** tidak otomatis tersimpan (akan hilang jika ada event listener)
- **Keamanan**: Rentan terhadap XSS (Cross-Site Scripting) jika tidak hati-hati

<a name="penjelasan-step-by-step-metode-2"></a>
### ğŸ” Penjelasan Step-by-Step Metode 2

#### **Step 1: Membuat Element `<li>`**

- Sama seperti metode 1, membuat elemen list item baru

#### **Step 2: Menambahkan Text Node**

- `document.createTextNode(item)` = membuat **text node** (bukan elemen HTML!)
- Text node ini berisi teks item (contoh: "Cheese")
- `li.appendChild()` = menambahkan text node ke dalam `<li>`

> ğŸ’¡ **Text Node** adalah teks murni tanpa tag HTML

#### **Step 3: Membuat Element `<button>`**

- Membuat elemen button baru
- `button.className` = menambahkan class CSS ke button

---

#### **Step 4: Membuat Element `<i>` (Icon)**
```javascript

icon.className = 'fa-solid fa-xmark';
```

- Membuat elemen `<i>` untuk icon Font Awesome
- Menambahkan class untuk menampilkan icon X

#### **Step 5: Memasukkan Icon ke dalam Button**
```javascript

```
- Icon `<i>` dimasukkan ke dalam `<button>`

**Hasil sementara:**
```html

</button>
```

---

#### **Step 6: Memasukkan Button ke dalam `<li>`**
```javascript

```
- Button (yang sudah berisi icon) dimasukkan ke dalam `<li>`

**Hasil sementara:**

```html
  Cheese

    <i class="fa-solid fa-xmark"></i>
  </button>
```

#### **Step 7: Menambahkan ke DOM**

## ğŸ’¡ Tips untuk Pemula

### ğŸš« **Kesalahan Umum:**

#### âŒ **SALAH - Menggunakan String sebagai Node**
```javascript

```

#### âœ… **BENAR - Membuat Node dengan createElement**

---

### ğŸ“š **Konsep Penting:**

1. **Node vs String**
   - Node = Objek JavaScript yang merepresentasikan elemen HTML
   - String = Teks biasa

2. **appendChild() butuh Node**
   - Tidak bisa memasukkan string
   - Harus menggunakan `createElement()` atau `createTextNode()`

3. **innerHTML vs appendChild**
   - `innerHTML` = menulis HTML sebagai string
   - `appendChild` = menambahkan node yang sudah dibuat
