## ğŸš€ Memasukkan Element ke Halaman

Ada 2 method untuk memasukkan element ke halaman: cara lama (`appendChild`) dan cara modern (`append`).

<a name="method-appendchild"></a>
### ğŸ”µ Method: `appendChild()` (Cara Tradisional)

Ini method yang diajarkan di video (cara lama tapi masih valid):

**Syntax:**
```javascript

```

**Contoh:**
```javascript

div.textContent = 'Hello World';

// Masukkan ke body

```

**Karakteristik:**
- âœ… Hanya bisa memasukkan **1 element** per pemanggilan
- âœ… Harus berupa **Node** (tidak bisa string langsung)
- âœ… **Return** element yang di-append
- âš ï¸ Lebih verbose untuk banyak element

---

<a name="method-append"></a>
### ğŸŸ¢ Method: `append()` (Modern & Recommended) â­

Ini method yang **lebih modern** dan sering digunakan di tutorial terbaru:

**Syntax:**
```javascript

```

**Contoh 1: Append 1 Element**
```javascript

// Masukkan ke body

```

**Contoh 2: Append Banyak Element Sekaligus** ğŸ”¥
```javascript

div2.textContent = 'Item 2';

```

**Contoh 3: Append String Langsung** ğŸ”¥
```javascript

```


**âœ… Kelebihan:**
- Bisa append **banyak element sekaligus**
- Bisa append **string langsung** tanpa `createTextNode()`
- Lebih **fleksibel** dan **praktis**
- Kode lebih **ringkas**

**âš ï¸ Kekurangan:**
- Tidak return apa-apa (tidak bisa chaining)
