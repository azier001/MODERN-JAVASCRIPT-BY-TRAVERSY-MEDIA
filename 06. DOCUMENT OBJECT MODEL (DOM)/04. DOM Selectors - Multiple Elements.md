### ğŸ“¦ Apa itu NodeList?

**NodeList** adalah struktur data seperti array yang berisi kumpulan element DOM.

```javascript

// Output: NodeList(4) [li.item, li.item, li.item, li.item]
```

**Karakteristik NodeList:**
- âœ… Mirip seperti array
- âœ… Bisa diakses menggunakan index `[0]`, `[1]`, dll
- âœ… Bisa menggunakan `.forEach()` langsung
- âœ… Memiliki property `.length`

### ğŸ¯ Mengakses Element Berdasarkan Index

Seperti array, kita bisa mengakses element tertentu menggunakan **bracket notation** `[index]`.

<a name="menggunakan-foreach-pada-nodelist"></a>
### ğŸ”„ Menggunakan forEach() pada NodeList

Untuk mengubah **semua element sekaligus**, gunakan `forEach()`:

```javascript

// Mengubah semua item menjadi merah

  item.style.color = 'red';
});
```

**Penjelasan:**
- `.forEach()` - method untuk loop/iterasi setiap element
- `(item)` - parameter yang merepresentasikan element saat ini dalam loop
- `item.style.color = 'red'` - mengubah warna setiap item menjadi merah

**âœ… Hasil:** Semua list items akan berwarna merah!

### ğŸ”§ Memanipulasi Element Berdasarkan Kondisi

**Penjelasan:**

**Penjelasan:**
- `Array.from()` - method built-in JavaScript untuk convert array-like object ke array
- Setelah diconvert, semua array methods bisa digunakan! ğŸ‰

<a name="tips-dan-best-practices"></a>
## ğŸ’¡ Tips dan Best Practices

### âœ… Do's (Yang Benar):

1. **Gunakan `querySelectorAll()` untuk semua kasus**
   ```javascript

   ```

2. **Gunakan `forEach()` untuk loop multiple elements**
   ```javascript
   items.forEach((item) => {
     item.style.color = 'red';
   });
   ```

3. **Gunakan `innerHTML` jika perlu preserve child elements**
   ```javascript
   item.innerHTML = `New Text <button>Click</button>`;
   ```

4. **Convert HTMLCollection ke Array jika diperlukan**
   ```javascript
   const itemsArray = Array.from(htmlCollection);
   ```

---

### âŒ Don'ts (Yang Salah):

1. **Jangan apply style langsung ke NodeList/HTMLCollection**
   ```javascript
   // âŒ SALAH
   items.style.color = 'red'; // Error!
   
   // âœ… BENAR
   items.forEach(item => item.style.color = 'red');
   ```

2. **Jangan gunakan `getElementsByClassName()` atau `getElementsByTagName()` kecuali terpaksa**
   ```javascript
   // âš ï¸ Method lama
   const items = document.getElementsByClassName('item');
   
   // âœ… Lebih baik
   const items = document.querySelectorAll('.item');
   ```

3. **Jangan lupa tentang child elements saat update content**
   ```javascript
   // âŒ Akan hapus button
   item.innerText = 'New Text';
   
   // âœ… Preserve button
   item.innerHTML = 'New Text <button>Delete</button>';
   ```
