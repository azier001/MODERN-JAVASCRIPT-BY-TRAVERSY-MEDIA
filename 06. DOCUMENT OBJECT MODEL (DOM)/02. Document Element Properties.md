## ğŸ”„ HTMLCollection vs Array

### Perbedaan Utama

**HTMLCollection**:
- ğŸ“¦ Objek khusus yang mirip array
- âœ… Bisa diakses dengan index: `collection[0]`
- âŒ TIDAK bisa menggunakan method array seperti `forEach()`

**Array**:
- ğŸ“‹ Array JavaScript biasa
- âœ… Bisa menggunakan semua method array
- âœ… Bisa menggunakan `forEach()`, `map()`, `filter()`, dll

### Masalah dengan forEach

```javascript

  console.log(form);
});
// Error: document.forms.forEach is not a function
```

**Mengapa Error?**
- ğŸš« `document.forms` mengembalikan HTMLCollection
- ğŸš« HTMLCollection BUKAN array
- ğŸš« Tidak memiliki method `forEach()`

### Solusi: Konversi ke Array

```javascript

// Sekarang bisa menggunakan forEach

  console.log(form);
});
// Output: <form id="item-form">...</form>
```

**Penjelasan `Array.from()`**:
- ğŸ”„ Method yang mengubah array-like object menjadi array
- âœ… Setelah dikonversi, bisa pakai semua method array
- ğŸ’¡ Sangat berguna untuk HTMLCollection dan NodeList

### Contoh Lengkap

```javascript

console.log(formsCollection); // HTMLCollection

// 2. Konversi ke array

console.log(formsArray); // Array [form]

// 3. Sekarang bisa pakai forEach, map, filter, dll

  console.log(form.id); // Output: "item-form"
  console.log(form.method); // Output: "get"
});
```

### Catatan Penting

ğŸ’¡ **NodeList vs HTMLCollection**:
- `querySelector()` mengembalikan **NodeList** (bisa pakai forEach)
- `document.forms`, `document.images`, dll mengembalikan **HTMLCollection** (tidak bisa pakai forEach)
