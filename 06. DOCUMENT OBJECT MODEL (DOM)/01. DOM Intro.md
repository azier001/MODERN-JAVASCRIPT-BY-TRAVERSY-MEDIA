## ğŸ“– Apa itu DOM?

### Definisi Sederhana

**DOM (Document Object Model)** adalah jembatan antara HTML, CSS, dan JavaScript. Ini adalah cara JavaScript untuk "berbicara" dengan halaman web Anda.

Bayangkan DOM seperti ini:
- ğŸ  **HTML** = Struktur rumah (dinding, pintu, jendela)
- ğŸ¨ **CSS** = Dekorasi rumah (warna, ukuran, tata letak)
- âš™ï¸ **JavaScript + DOM** = Sistem otomatis rumah (membuka pintu, menyalakan lampu)

### Apa yang Termasuk dalam DOM?

DOM tidak hanya mencakup tag HTML saja, tetapi juga:

```
1. ğŸ·ï¸ Tag HTML (seperti <div>, <h1>, <p>)
2. ğŸ“ Text nodes (teks di antara tag)
3. ğŸ”§ Attributes (atribut seperti id, class, href)
4. ğŸ¯ Seluruh struktur halaman
```

### Mengapa DOM Penting?

Dengan DOM, JavaScript dapat:
- âœ… Membaca isi halaman
- âœ… Mengubah teks dan HTML
- âœ… Menambah atau menghapus elemen
- âœ… Merespons aksi pengguna (klik, ketik, dll)
- âœ… Membuat halaman interaktif dan dinamis

## ğŸ” Mengakses DOM dengan Properties

### Method 1: Menggunakan Properties Langsung

Document memiliki beberapa properti built-in yang mudah diakses:

#### Mengakses Body

```javascript

// Output: Menampilkan seluruh isi <body>
```

#### Mengakses Semua Link

```javascript
// Mengakses semua tag <a> dalam halaman

// Output: HTML Collection (mirip array) berisi semua link
// Bisa diakses dengan indeks:

console.log(document.links[1]);  // Link kedua
console.log(document.links[2]);  // Link ketiga
```

#### Mengakses Semua Gambar

```javascript
// Mengakses semua tag <img> dalam halaman
console.log(document.images);

// Mirip dengan links, bisa diakses dengan indeks:
console.log(document.images[0]);  // Gambar pertama
```

## ğŸ¯ Mengakses DOM dengan Methods

### Method 1: getElementById() - Mencari Berdasarkan ID

Ini adalah cara yang paling spesifik untuk mencari satu elemen.

```javascript

```

**Kapan Digunakan?**
- âœ… Ketika ingin mengakses satu elemen spesifik
- âœ… Ketika elemen memiliki ID unik

---

### Method 2: querySelector() - Pencari Fleksibel (â­ RECOMMENDED)

Ini adalah method yang paling powerful dan fleksibel. Bisa mencari dengan ID, class, atau tag apapun!

```javascript

// Mencari berdasarkan class (gunakan .)

// Mencari berdasarkan tag langsung

// Mencari dengan kombinasi (H1 di dalam #main)

// Semua format CSS selector berfungsi!
```

**Keuntungan querySelector():**
- ğŸ’ª Sangat fleksibel
- ğŸ“ Menggunakan sintaks CSS (familiar!)
- ğŸ¯ Bisa kombinasi selector
- â­ Recommended untuk pemula

---

### Method 3: querySelectorAll() - Mencari Banyak Elemen

Ketika ingin mengakses lebih dari satu elemen.

```javascript

// Mengakses yang pertama

// Mengakses yang kedua
```

Gunakan `innerHTML` untuk mengganti HTML:


**Penting!** âš ï¸ Ini akan menghapus semua konten sebelumnya dan menggantinya!

```html

    <h1>Halo Dunia!</h1>
```

---

### Mengubah Hanya Teks (Lebih Aman)

Gunakan `innerText` atau `textContent` untuk mengubah hanya teks:

```javascript
// Mengubah hanya teks dalam H1
document.querySelector('h1').innerText = 'Teks Baru';

// Atau dengan textContent (hampir sama)
document.querySelector('h1').textContent = 'Teks Baru';
```

**Perbedaan innerText vs textContent:**

| innerText | textContent |
|-----------|-------------|
| Hanya menampilkan teks yang terlihat | Menampilkan semua teks |
| Respek terhadap CSS styling | Tidak peduli CSS |
| Sedikit lebih lambat | Lebih cepat |

Untuk pemula, gunakan `innerText` - lebih intuitif! ğŸ‘
