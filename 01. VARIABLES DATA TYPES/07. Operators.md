## ğŸ’¡ Tips untuk Pemula

### âœ… **Best Practices:**

1. **Gunakan `===` bukan `==`**
   - Lebih aman dan menghindari bug
   - Mengecek tipe data juga

2. **Gunakan operator shorthand untuk efisiensi**
   - `x += 5` lebih singkat dari `x = x + 5`
   - Kode lebih bersih dan mudah dibaca

3. **Hati-hati dengan tipe data**
   - `5 + 5` = `10` (number)
   - `'5' + '5'` = `'55'` (string)

### âš ï¸ **Kesalahan Umum:**

  // ...
