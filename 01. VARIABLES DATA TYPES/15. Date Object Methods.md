**ğŸ” Perbedaan dengan Math Object:**

## ğŸŒ Intl.DateTimeFormat API

### Apa itu Intl.DateTimeFormat?

API modern untuk memformat tanggal sesuai dengan **locale** (lokasi/negara) tertentu.

### 1ï¸âƒ£ Format Dasar

**ğŸ—ºï¸ Locale Options:**

- `default` = Otomatis sesuai lokasi browser

### 2ï¸âƒ£ Format dengan Options

**âš™ï¸ Options yang Tersedia:**

| `month` | `'long'`, `'short'`, `'numeric'` | October, Oct, 10 |
| `day` | `'numeric'`, `'2-digit'` | 28, 28 |
| `hour` | `'numeric'`, `'2-digit'` | 9, 09 |
| `minute` | `'numeric'`, `'2-digit'` | 49, 49 |
