## âš ï¸ Perbedaan Kunci

| Aspek | Primitive Types | Reference Types |
|-------|----------------|-----------------|
| **Penyimpanan** | Stack (nilai langsung) | Heap (via referensi) |
| **Copy Behavior** | Membuat nilai baru | Membuat referensi baru |
| **Perubahan** | Tidak mempengaruhi variable lain | Mempengaruhi semua variable yang mereferensi |
| **Contoh** | `string`, `number`, `boolean` | `object`, `array`, `function` |

## ğŸ“ Kesimpulan

- **Primitive types** disimpan di **Stack** dengan nilai langsung
- **Reference types** disimpan di **Heap** dan diakses via referensi
- Meng-copy primitive types membuat nilai independen baru
- Meng-copy reference types membuat pointer ke data yang sama
- Mengubah reference type akan mempengaruhi semua variable yang mereferensinya
