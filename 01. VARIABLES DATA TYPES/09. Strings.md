## ğŸ¯ Pengenalan String

**String** adalah tipe data primitif di JavaScript yang digunakan untuk menyimpan teks. Meskipun primitif, JavaScript secara otomatis memberikan "wrapper" (pembungkus) ketika kita menggunakan method atau property pada string, sehingga kita bisa mengakses berbagai fungsi yang sangat berguna.

### ğŸ’¡ Fakta Penting:
- String adalah **tipe data primitif**
- JavaScript otomatis mengubahnya menjadi **object** saat kita pakai method
- String bisa dibuat dengan `''`, `""`, atau `` ` ` `` (backticks)

## ğŸ”— Concatenation (Penggabungan String)

Concatenation adalah cara **menggabungkan** beberapa string menjadi satu menggunakan operator `+`.

### ğŸ“ Contoh Kode:

```javascript

```

### âš ï¸ Kekurangan Concatenation:

- Tidak praktis untuk string yang kompleks

## âœ¨ Template Literals (ES6)

Template Literals adalah cara **modern** dan **lebih mudah** untuk membuat string yang berisi variabel atau ekspresi JavaScript. Diperkenalkan di **ES2015 (ES6)**.

### ğŸ¯ Cara Menggunakan:
- Gunakan **backticks** `` ` ` `` (bukan tanda kutip biasa)

### ğŸ“ Contoh Kode:


### ğŸš€ Keunggulan Template Literals:
- âœ… Lebih mudah dibaca

### ğŸ’¡ Contoh dengan Ekspresi:

Property adalah **atribut** dari sebuah string. Property **tidak menggunakan tanda kurung ()**.

### ğŸ“ `.length` - Menghitung Panjang String


Method adalah **fungsi** yang bisa dijalankan pada string. Method **menggunakan tanda kurung ()**.

### ğŸ”¤ 1. `.toUpperCase()` dan `.toLowerCase()`

Mengubah huruf menjadi **KAPITAL** atau **kecil**.

```javascript

// Ubah ke UPPERCASE

console.log(x);
// Output: hello world
```

**ğŸ’¡ Kapan digunakan?**
- Membandingkan string tanpa peduli huruf besar/kecil
- Validasi input user

---

### ğŸ¯ 2. `.charAt(index)`

Mengembalikan **karakter** pada posisi tertentu.

```javascript

// Output: H
```

**ğŸ“Œ Catatan:** Sama seperti menggunakan `s[0]`, tapi lebih eksplisit.

### ğŸ” 3. `.indexOf(searchValue)`

Mencari **posisi pertama** dari karakter/substring dan mengembalikan **index**-nya.

**âš ï¸ Penting:** Jika tidak ditemukan, akan return `-1`

### âœ‚ï¸ 4. `.substring(start, end)`

Mengambil **bagian dari string** dari index `start` sampai **sebelum** index `end`.

**ğŸ“Œ Ingat:** Index dimulai dari **0**

**ğŸ’¡ Perbedaan dengan substring:**

### ğŸ’ 9. `.valueOf()`

Mengembalikan **nilai primitif** dari string object.

**ğŸ“Œ Info:** Biasanya JavaScript otomatis handle ini, jarang digunakan manual.

---

### ğŸ“¦ 10. `.split(separator)`

Memecah string menjadi **array** berdasarkan separator.

```javascript

// Split berdasarkan spasi
let x = s.split(' ');
console.log(x);
// Output: ['Hello', 'World']

// Split setiap karakter

console.log(x);
// Output: ['H', 'e', 'l', 'l', 'o', ' ', 'W', 'o', 'r', 'l', 'd']

// Tanpa separator - jadi 1 element array

console.log(x);
// Output: ['Hello World']
```

**ğŸ¯ Sangat berguna untuk:**
- Memproses CSV
- Parsing data
- Manipulasi teks

## ğŸ“ Kesimpulan

### ğŸ“ Ringkasan Penting:

| **Fitur** | **Kegunaan** | **Contoh** |
|-----------|--------------|------------|
| **Concatenation** | Gabung string dengan `+` | `'Hello ' + name` |
| **Template Literals** | Gabung string modern | `` `Hello ${name}` `` |
| `.length` | Hitung panjang | `s.length` |
| `.toUpperCase()` | Huruf besar semua | `s.toUpperCase()` |
| `.toLowerCase()` | Huruf kecil semua | `s.toLowerCase()` |

| `.slice()` | Potong (bisa negatif) | `s.slice(-5)` |
| `.trim()` | Hapus spasi pinggir | `s.trim()` |
| `.replace()` | Ganti teks | `s.replace('a', 'b')` |
| `.includes()` | Cek ada/tidak | `s.includes('Hello')` |
| `.split()` | Pecah jadi array | `s.split(' ')` |

### ğŸŒŸ Tips untuk Pemula:

1. **Gunakan Template Literals** untuk string yang kompleks
2. **Index dimulai dari 0**, jangan lupa!
3. String di JavaScript **immutable** (tidak bisa diubah), method selalu return string baru
4. Gunakan `.trim()` saat menerima input dari user
5. `.includes()` lebih mudah dibaca daripada `.indexOf() !== -1`
