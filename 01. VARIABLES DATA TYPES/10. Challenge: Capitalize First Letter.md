## ðŸ†š Perbandingan `substring()` vs `slice()`

| Aspek | `substring(start)` | `slice(start)` |
|-------|-------------------|----------------|
| **Fungsi** | Ambil sebagian string | Ambil sebagian string |
| **Syntax** | `string.substring(1)` | `string.slice(1)` |
| **Negative Index** | âŒ Tidak support | âœ… Support |
| **Contoh** | `'developer'.substring(1)` = `'eveloper'` | `'developer'.slice(1)` = `'eveloper'` |
| **Modern?** | Legacy method | âœ… Lebih modern |

### Contoh Perbedaan dengan Negative Index:


> ðŸ’¡ **Rekomendasi:** Gunakan `slice()` karena lebih powerful dan konsisten!

## ðŸ“Š Method Chaining Explained

**Method chaining** adalah teknik memanggil multiple methods secara berurutan:

```javascript
// Tanpa chaining (verbose)

const upperChar = firstChar.toUpperCase();

// Dengan chaining (concise) âœ…
const upperChar = myString.charAt(0).toUpperCase();
```

### ðŸ”— Cara Kerja Chaining:

```javascript

//       â†“            â†“
//     return 'd'   return 'D'
```

1. `myString.charAt(0)` â†’ return `'d'` (string)
2. `'d'.toUpperCase()` â†’ return `'D'` (string)

> âš ï¸ **Penting:** Method chaining hanya bisa dilakukan jika method sebelumnya **return value yang memiliki method** tersebut!
