<a name="apa-itu-function"></a>
## ğŸ“– Apa itu Function?

**Function** adalah blok kode yang dapat digunakan kembali untuk melakukan tugas tertentu. Function dapat menerima input (parameter) dan mengembalikan output (return value).

**Contoh sederhana:**

**Keuntungan menggunakan Function:**

<a name="apa-itu-switch-statement"></a>
## ğŸ”€ Apa itu Switch Statement?

**Switch Statement** adalah struktur kontrol yang digunakan untuk menjalankan kode berbeda berdasarkan nilai tertentu. Ini adalah alternatif dari multiple `if-else` statements.

**Struktur dasar:**
```javascript
switch (ekspresi) {
  case nilai1:
    // kode jika ekspresi === nilai1
    break;
  case nilai2:
    // kode jika ekspresi === nilai2
    break;
  default:
    // kode jika tidak ada case yang cocok
}
```

**Kapan menggunakan Switch?**
- âœ¨ Ketika membandingkan satu nilai dengan banyak kemungkinan
- âœ¨ Membuat kode lebih readable dibanding banyak if-else
- âœ¨ Untuk kasus-kasus yang mutually exclusive (hanya satu yang akan dijalankan)

<a name="step-4-menambahkan-case-untuk-setiap-operator"></a>
### Step 4: Menambahkan Case untuk Setiap Operator â•â–âœ–ï¸â—

Tambahkan case untuk setiap operator matematika:

```javascript
function calculator(num1, num2, operator) {
  let result;
  
  switch (operator) {
    case '+':
      result = num1 + num2;
      break;
    case '-':
      result = num1 - num2;
      break;
    case '*':
      result = num1 * num2;
      break;
    case '/':
      result = num1 / num2;
      break;
  }
}
```

**Penjelasan Detail:**
- `case '+'` = jika operator adalah string '+'
- `result = num1 + num2` = lakukan penjumlahan
- `break` = **PENTING!** Keluar dari switch setelah case ini selesai
- Tanpa `break`, kode akan terus ke case berikutnya (fall-through)

**âš ï¸ Catatan Penting:**
- Operator harus dalam **tanda kutip** karena berupa **string**
- Jangan lupa `break` di setiap case!

<a name="step-5-menambahkan-default-case"></a>
### Step 5: Menambahkan Default Case âš ï¸

Tambahkan `default` case untuk handle operator yang tidak valid:

  switch (operator) {

      result = num1 + num2;
      break;
    case '-':
      result = num1 - num2;

    case '*':
      result = num1 * num2;

      result = num1 / num2;

```

**Penjelasan:**
- `default` = dijalankan jika tidak ada case yang cocok
- Seperti `else` dalam if-else statement
- Tidak perlu `break` karena sudah di akhir switch

## ğŸ” Penjelasan Detail Kode

### 1ï¸âƒ£ Flow Eksekusi

Ketika kita memanggil `calculator(5, 2, '+')`:

```

---

### 2ï¸âƒ£ Kenapa Pakai `let result;`?

**Alasan 1: Scope**
- Variable `result` perlu diakses di luar switch statement
- Jika dibuat di dalam case, tidak bisa diakses di luar

**Alasan 2: Fleksibilitas**
- Setiap case bisa mengubah nilai `result`
- Value-nya ditentukan berdasarkan kondisi

**Contoh SALAH:**
```javascript
switch (operator) {
  case '+':
    let result = num1 + num2; // âŒ Scope terbatas di case ini
    break;
}
console.log(result); // âŒ Error: result is not defined
```

**Contoh BENAR:**
```javascript
let result; // âœ… Dibuat di luar switch
switch (operator) {
  case '+':
    result = num1 + num2; // âœ… Mengisi variable yang sudah ada
    break;
}
console.log(result); // âœ… Bisa diakses
```

---

### 3ï¸âƒ£ Pentingnya `break` Statement

**Dengan `break`:**
```javascript
switch (operator) {
  case '+':
    result = num1 + num2;
    break; // âœ… Keluar dari switch
  case '-':
    result = num1 - num2;
    break;
}
```

**Tanpa `break` (WRONG!):**
```javascript
switch (operator) {
  case '+':
    result = num1 + num2;
    // âŒ Tidak ada break, terus ke case berikutnya!
  case '-':
    result = num1 - num2; // Ini juga akan dieksekusi!
    break;
}
```

**Hasil tanpa break:**
```javascript
calculator(5, 2, '+');
// Hasilnya: 3 (SALAH!)
// Seharusnya: 7
// Yang terjadi: 
// 1. result = 5 + 2 = 7
// 2. Tidak ada break, lanjut ke case '-'
// 3. result = 5 - 2 = 3 (menimpa nilai sebelumnya!)
```

---

### 4ï¸âƒ£ Default Case

`default` case seperti **safety net** atau jaring pengaman:

```javascript
default:
  result = 'Invalid Operator';
```

**Kapan default dijalankan?**
- Ketika operator bukan '+', '-', '*', atau '/'
- Contoh: '&', '%', '^', 'abc', dll

**Alternatif pesan error:**
```javascript
default:
  result = 'Error: Please use +, -, *, or /';
  // atau
  result = `Operator '${operator}' is not supported`;
```

### âœ¨ Best Practices

### ğŸ”„ Alternatif: Menggunakan If-Else

Kamu juga bisa menggunakan if-else statement:

**Switch vs If-Else:**

| Switch Statement | If-Else Statement |
|-----------------|-------------------|
| âœ… Lebih clean untuk banyak kondisi | âœ… Lebih fleksibel untuk kondisi kompleks |
| âœ… Lebih readable | âš ï¸ Bisa jadi verbose |
| âœ… Performance sedikit lebih baik | âš ï¸ Harus menulis operator === berkali-kali |
| âš ï¸ Hanya untuk equality check | âœ… Bisa untuk range check (>, <, dll) |
