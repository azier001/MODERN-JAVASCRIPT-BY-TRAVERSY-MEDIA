## ğŸ“– Apa itu Logical Assignment Operators?

**Logical Assignment Operators** adalah operator khusus yang menggabungkan **operator logika** dengan **operator assignment** (=).

Ada 3 jenis operator ini:

| Operator | Nama | Fungsi |
|----------|------|--------|
| `\|\|=` | OR Assignment | Mengisi nilai jika variabel **falsy** |
| `&&=` | AND Assignment | Mengisi nilai jika variabel **truthy** |
| `??=` | Nullish Coalescing Assignment | Mengisi nilai jika variabel **null/undefined** |

> ğŸ’¡ **Catatan Penting:**  
> - **Falsy values**: `false`, `0`, `""` (string kosong), `null`, `undefined`, `NaN`
> - **Truthy values**: Semua nilai selain falsy values

## ğŸ¤” Kapan Menggunakan Operator Ini?

### âœ… Gunakan `||=` ketika:
- Setting nilai default untuk variabel
- Ingin mengganti semua nilai falsy
- Contoh: default parameter, fallback values

```javascript

username ||= "Guest";  // Jadi "Guest"

let score = 0;
score ||= 100;  // Jadi 100
```

### âœ… Gunakan `&&=` ketika:
- Update nilai hanya jika sudah ada
- Conditional update
- Contoh: mengubah data yang sudah terisi

```javascript

loggedInUser &&= "John Doe";  // Update ke "John Doe"

let notLoggedIn = null;
notLoggedIn &&= "John Doe";  // Tetap null
```

### âœ… Gunakan `??=` ketika:
- Nilai `0`, `false`, atau `""` adalah nilai yang valid
- Hanya ingin mengisi jika benar-benar tidak ada data
- Contoh: form input, API response

```javascript
let age = 0;  // 0 adalah umur yang valid
age ??= 18;  // Tetap 0

let name = null;  // Benar-benar tidak ada data
name ??= "Anonymous";  // Jadi "Anonymous"
```
