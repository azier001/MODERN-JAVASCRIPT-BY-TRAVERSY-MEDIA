## ğŸŒ¿ Else If Statement

<a name="apa-itu-else-if"></a>
### â“ Apa itu Else If?

**Else If** adalah cara untuk mengevaluasi **beberapa kondisi secara berurutan**. Ketika kondisi pertama tidak terpenuhi, JavaScript akan mengecek kondisi selanjutnya, dan seterusnya.

**Struktur dasar:**
```javascript
if (kondisi1) {
  // Jalankan jika kondisi1 true
} else if (kondisi2) {
  // Jalankan jika kondisi1 false dan kondisi2 true
} else {
  // Jalankan jika semua kondisi false
}
```

---

<a name="cara-kerja-else-if"></a>
### âš™ï¸ Cara Kerja Else If

**Alur eksekusi:**

1. âœ… Cek **kondisi pertama** â†’ Jika `true`, jalankan block code-nya dan **BERHENTI**
2. âŒ Jika `false`, lanjut ke **else if** â†’ Jika `true`, jalankan block code-nya dan **BERHENTI**
3. âŒ Jika `false` lagi, lanjut ke **else** â†’ Jalankan block code terakhir

> **âš ï¸ PENTING:** Begitu satu kondisi terpenuhi, kondisi-kondisi selanjutnya **TIDAK AKAN DICEK**!

## ğŸª† Nested If Statement

<a name="apa-itu-nested-if"></a>
### â“ Apa itu Nested If?

**Nested If** adalah **if statement di dalam if statement** lainnya. Ini berguna ketika kita ingin mengevaluasi kondisi tambahan **hanya jika** kondisi pertama sudah terpenuhi.

**Struktur dasar:**
```javascript
if (kondisi_luar) {
  // Code untuk kondisi luar
  
  if (kondisi_dalam) {
    // Code untuk kondisi dalam
  }
}
```

<a name="contoh-praktik-nested-if"></a>
### ğŸ’» Contoh Praktik Nested If

#### **Contoh 2: Ucapan dengan Pesan Tambahan**

**ğŸ¯ Penjelasan Langkah demi Langkah:**

**Skenario 2: Jam = 9**

**Skenario 3: Jam = 21**

2. Cek `hour < 18` â†’ `21 < 18` = false âŒ

6. Cetak "zzzzzzzz"

**Output:**
```

```

**ğŸ“Œ Kapan Menggunakan Nested If?**

âœ… Gunakan ketika kondisi kedua **hanya relevan** jika kondisi pertama terpenuhi

**ğŸ¯ Penjelasan:**

## ğŸ’¡ Tips & Best Practices

### âœ… DO (Lakukan)

   ```javascript
   }
   ```

   ```javascript

   }
   ```


### âŒ DON'T (Jangan)

   ```javascript

           // Susah dibaca!
         }
       }
     }
   }
   ```

   ```javascript
   // âœ… LEBIH BAIK - Gunakan nested if
   ```
