### â“ Pertanyaan Penting

### ğŸ’¡ Penjelasan Local Storage

#### ğŸ” **Perbedaan Server vs Browser:**

**Server (Netlify):**
- ğŸ–¥ï¸ Menyimpan **file aplikasi** (HTML, CSS, JS)
- ğŸŒ Mengirimkan file ke browser pengguna

- ğŸ’» Menjalankan aplikasi

- ğŸ”’ Data tersimpan **lokal** di komputer user

#### ğŸ“Š **Ilustrasi Cara Kerja:**

```

        â†“
Server Netlify mengirim file (HTML, CSS, JS)
        â†“
Browser user menjalankan aplikasi
        â†“
User menambahkan item "juice"
        â†“
JavaScript menyimpan "juice" ke Local Storage BROWSER
        â†“

```

### âœ¨ **Kesimpulan:**

- âœ… Local Storage bekerja di **browser** (client-side)
- âœ… Data tersimpan di **komputer user**, bukan di server
