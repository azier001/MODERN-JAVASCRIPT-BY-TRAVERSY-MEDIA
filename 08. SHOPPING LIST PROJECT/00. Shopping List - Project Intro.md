## ğŸ§  Konsep Penting yang Dipelajari

### 1. ğŸŒ DOM (Document Object Model)

### 3. ğŸ’¾ Local Storage

**Data Type:**
Local storage hanya bisa simpan **string**, makanya kita perlu:
- `JSON.stringify()` - Ubah array â†’ string (sebelum simpan)
- `JSON.parse()` - Ubah string â†’ array (setelah load)

### 4. ğŸ¨ Dynamic UI

**Contoh di Project Ini:**
