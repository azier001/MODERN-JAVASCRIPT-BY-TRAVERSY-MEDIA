## ğŸ­ Konsep Event Delegation

### â“ Apa itu Event Delegation?

**Event Delegation** adalah teknik di mana kita menaruh event listener pada **parent element** (elemen induk), bukan pada setiap child element (elemen anak).

### ğŸ¤” Kenapa Menggunakan Event Delegation?

- Kita punya **banyak tombol âŒ** (satu untuk setiap item)
- Daripada membuat event listener untuk **setiap tombol**, lebih efisien taruh event listener di **parent** (dalam kasus ini: `<ul>`)
- Kemudian kita **target** elemen spesifik yang diklik

### ğŸ“Š Struktur DOM Kita

```

  <li>                           â† List Item
```

<a name="metode-while-loop-recommended"></a>
### 3ï¸âƒ£ Metode While Loop (Recommended) â­

Cara yang lebih baik dan lebih cepat:

```javascript

  while (itemList.firstChild) {
    itemList.removeChild(itemList.firstChild);
  }
}
```

**Penjelasan Step by Step:**

1. **`while (itemList.firstChild)`** â†’ Selama `<ul>` masih punya child pertama
2. **`itemList.removeChild(itemList.firstChild)`** â†’ Hapus child pertama
3. Loop terus berulang sampai tidak ada child lagi

**Analogi:**
- Bayangkan kamu punya tumpukan piring
- Ambil piring paling atas, buang
- Ambil piring paling atas lagi, buang
- Ulangi sampai tidak ada piring tersisa

**Kenapa lebih cepat?**
- Langsung manipulasi DOM node
- Tidak perlu parse HTML seperti `innerHTML`
- Lebih efisien untuk memory

> ğŸ’¡ **Best Practice**: Gunakan metode ini untuk menghapus semua child element!

### ğŸ“ Apa yang Sudah Dipelajari?

1. **Event Delegation** ğŸ­
   - Taruh event listener di parent

   - Lebih efisien untuk banyak element

2. **Event Target & Parent Element** ğŸ¯
   - `e.target` = element yang diklik
   - `parentElement` = naik ke parent
   - Bisa chain multiple `parentElement`

3. **Class List & Contains** ğŸ“
   - `classList` = daftar class pada element
   - `contains('class-name')` = cek apakah class ada

4. **Menghapus Element** ğŸ—‘ï¸
   - `element.remove()` = hapus element
   - `parent.removeChild(child)` = hapus child dari parent

5. **While Loop untuk Clear All** ğŸ§¹
   - Loop selama ada `firstChild`
   - Hapus `firstChild` berulang kali
   - Lebih efisien dari `innerHTML`
