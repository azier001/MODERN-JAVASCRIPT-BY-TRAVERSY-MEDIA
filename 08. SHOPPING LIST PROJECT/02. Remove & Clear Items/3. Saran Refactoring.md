<a name="penjelasan-closest"></a>
## ğŸ” Penjelasan Method `.closest()`

### **Apa itu `.closest()`?**

Method `.closest()` adalah cara modern untuk **mencari elemen parent terdekat** yang cocok dengan selector CSS.

```javascript
element.closest('selector')
```

### **Ilustrasi Sederhana:**

Bayangkan struktur HTML seperti ini:

```html

  <li>                         <!-- Grandparent -->
    <button class="remove-item"> <!-- Parent -->
      <i class="fa-xmark"></i>   <!-- Kamu klik ini! -->
    </button>
  </li>
</ul>
```

**Ketika kamu klik ikon âŒ:**

```javascript

e.target.parentElement.parentElement // Dapat <li>

// Cara BARU (mudah!)

```

### **âœ¨ Keuntungan `.closest()`:**

| Fitur | Penjelasan |
|-------|------------|
| ğŸ¯ **Langsung ke tujuan** | Cari elemen yang kamu mau, tidak peduli berapa level jauhnya |
| ğŸ›¡ï¸ **Aman** | Kalau tidak ketemu, return `null` (tidak error) |

## ğŸ†š Perbandingan `.matches()` vs `.closest()`

### **Method `.matches()`**

Cek apakah elemen **itu sendiri** cocok dengan selector.

```javascript
if (e.target.matches('.fa-xmark')) {
  // Cek: Apakah yang diklik PERSIS ikon .fa-xmark?
}
```

### **Method `.closest()`**

Cari elemen terdekat (termasuk dirinya sendiri) yang cocok dengan selector.

```javascript
e.target.closest('li')
// Cari: Elemen <li> terdekat dari yang diklik
```

### **ğŸ“Š Perbandingan Detail:**

| Aspek | `.matches()` | `.closest()` |
|-------|--------------|--------------|
| **Fungsi** | Cek apakah elemen cocok | Cari elemen parent yang cocok |
| **Return** | `true` / `false` | Element atau `null` |
| **Use Case** | "Apakah ini yang diklik?" | "Cari parent yang cocok" |
| **Contoh** | `e.target.matches('.btn')` | `e.target.closest('.btn')` |

### **ğŸ¯ Kapan Pakai Mana?**

**Pakai `.matches()` jika:**
```javascript
// Kamu mau cek element yang diklik PERSIS apa
if (e.target.matches('.fa-xmark')) {
  console.log('Kamu klik ikon X!');
}
```

**Pakai `.closest()` jika:**
```javascript
// Kamu mau cari parent dari element yang diklik
const button = e.target.closest('.btn');
if (button) {
  console.log('Kamu klik button atau anak-anaknya!');
}
```
