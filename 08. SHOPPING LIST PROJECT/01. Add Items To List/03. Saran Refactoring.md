## ğŸ’» Kode Lengkap (Versi Final)

```javascript
// Mengambil elemen-elemen dari HTML
const itemForm = document.getElementById('item-form');
const itemInput = document.getElementById('item-input');
const itemList = document.getElementById('item-list');

/**
 * Fungsi untuk menambah item baru ke dalam list
 * Dipanggil saat form di-submit
 */
function addItem(e) {
  // Mencegah halaman reload saat form di-submit
  e.preventDefault();

  // Ambil nilai dari input dan hapus spasi di awal/akhir
  const newItem = itemInput.value.trim();

  // Validasi: cek apakah input kosong
  if (newItem === '') {
    alert('Please add an item');
    return;
  }

  // Buat item baru dan tambahkan ke list
  createListItem(newItem);

  // Kosongkan input setelah item ditambahkan
  itemInput.value = '';
}

/**
 * Fungsi untuk membuat elemen <li> lengkap dengan tombol hapus
 * Memisahkan logika pembuatan item agar lebih terorganisir
 */
function createListItem(itemText) {
  // Buat elemen <li>
  const li = document.createElement('li');
  
  // Isi <li> dengan text item
  li.textContent = itemText;

  // Buat tombol hapus
  const deleteButton = createDeleteButton();
  
  // Tambahkan tombol ke dalam <li> (lebih modern dari appendChild)
  li.append(deleteButton);

  // Tambahkan <li> ke dalam <ul>
  itemList.append(li);
}

/**
 * Fungsi untuk membuat tombol hapus (X)
 * Dipisah agar kode lebih modular dan mudah diubah
 */
function createDeleteButton() {
  // Buat elemen <button>
  const button = document.createElement('button');
  
  // Tambahkan class untuk styling (lebih fleksibel dari className)
  button.classList.add('remove-item', 'btn-link', 'text-red');

  // Buat icon X menggunakan Font Awesome
  const icon = document.createElement('i');
  icon.classList.add('fa-solid', 'fa-xmark');

  // Masukkan icon ke dalam button (lebih modern dari appendChild)
  button.append(icon);

  return button;
}

// Event Listener: jalankan fungsi addItem saat form di-submit
itemForm.addEventListener('submit', addItem);
```

```javascript
  return icon;
```

**Breakdown Step by Step:**

#### 2ï¸âƒ£ `const newItem = itemInput.value.trim()`

#### 3ï¸âƒ£ Validasi Input
```javascript

  alert('Please add an item');
```

- Cek apakah input kosong
- Jika kosong, tampilkan alert dan stop fungsi
- `return` = keluar dari fungsi

#### 4ï¸âƒ£ `createListItem(newItem)`
- Panggil fungsi lain untuk membuat item
- Prinsip: **Delegasi tugas ke fungsi yang tepat**

#### 5ï¸âƒ£ `itemInput.value = ''`
- Kosongkan input box setelah item ditambahkan
- User bisa langsung ketik item baru

- âœ… Lebih simpel dan modern
- âœ… Lebih mudah dibaca

| Fitur | appendChild() | append() |
|-------|---------------|----------|
| Jumlah elemen | 1 saja | Banyak sekaligus |
| Append text | âŒ Harus createTextNode | âœ… Langsung |
| Sintaks | Panjang | Pendek |
| Browser support | Semua | Modern (IE tidak) |
| Rekomendasi | â­â­ | â­â­â­â­â­ |

### ğŸ” const dan Value Assignment

#### â“ Pertanyaan yang Sering Muncul

**"Kenapa pakai `itemInput.value = ''` bukan `newItem = ''`?"**

**Jawaban:**

```javascript

// newItem = '';  // âŒ ERROR! const tidak bisa diubah
```

#### ğŸ“š Penjelasan Detail

**1ï¸âƒ£ `const newItem` adalah COPY (Read-Only)**


- `newItem` menyimpan **salinan** nilai input
- Seperti fotokopi KTP
- `const` = tidak bisa diubah nilainya


- `itemInput` adalah referensi ke elemen HTML asli
- `itemInput.value` = properti yang bisa diubah
- Seperti KTP asli yang bisa diganti

#### ğŸ” Perbedaan Value vs Reference

### 1ï¸âƒ£ **Single Responsibility Principle (SRP)**

> Satu fungsi hanya punya satu tugas

**Contoh dalam Kode:**

```javascript

// âœ… BAGUS: Setiap fungsi punya tugas spesifik
function addItem(e) {
```
