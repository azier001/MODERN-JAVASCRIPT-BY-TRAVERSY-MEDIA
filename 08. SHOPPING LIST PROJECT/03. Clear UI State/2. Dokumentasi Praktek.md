<a name="step-8-solusi-pindahkan-query-ke-dalam-fungsi"></a>
### ğŸ“ Step 8: Solusi - Pindahkan Query ke Dalam Fungsi

**ğŸ” Root Cause (Akar Masalah):**

Sebelumnya mungkin kamu punya kode seperti ini (DI GLOBAL SCOPE):
```javascript
const items = itemList.querySelectorAll('li'); // â† SALAH!

function checkUI() {
  if (items.length === 0) {
    // ...
  }
}
```

**Kenapa ini SALAH?**
- `const items` didefinisikan sekali di awal
- Ketika JavaScript dimuat, list masih kosong â†’ `items.length = 0`
- Value-nya **tidak akan berubah** walaupun kita add item!
- Karena variable `const` bersifat **read-only** setelah didefinisikan

**ğŸ’¡ Solusi:**
Pindahkan query `querySelectorAll()` **KE DALAM FUNGSI** checkUI():

```javascript
function checkUI() {
  // âœ… Query dilakukan SETIAP KALI fungsi dipanggil
  const items = itemList.querySelectorAll('li');

  if (items.length === 0) {
    clearBtn.style.display = 'none';
    itemFilter.style.display = 'none';
  } else {
    clearBtn.style.display = 'block';
    itemFilter.style.display = 'block';
  }
}
```

**Penjelasan:**
- Sekarang setiap kali `checkUI()` dipanggil
- `querySelectorAll('li')` dijalankan lagi
- Jadi selalu mendapat **data terbaru** dari DOM
- Length akan berubah sesuai jumlah item yang ada

```javascript
function createButton(classes) {

  button.className = classes;
  const icon = createIcon('fa-solid fa-xmark');
  button.appendChild(icon);
  return button;

function createIcon(classes) {
  const icon = document.createElement('i');
  icon.className = classes;
  return icon;
}

function removeItem(e) {
  if (e.target.parentElement.classList.contains('remove-item')) {
    if (confirm('Are you sure?')) {
      e.target.parentElement.parentElement.remove();

      checkUI(); // âœ… Update UI setelah delete

  }

    itemList.removeChild(itemList.firstChild);
  }

  // âœ… Query dilakukan di dalam fungsi (bukan global)
  const items = itemList.querySelectorAll('li');

  if (items.length === 0) {
    // ğŸ™ˆ List kosong â†’ Sembunyikan
    clearBtn.style.display = 'none';
    itemFilter.style.display = 'none';
  } else {
    // ğŸ‘€ Ada item â†’ Tampilkan
    clearBtn.style.display = 'block';
    itemFilter.style.display = 'block';
  }
}

// Event Listeners

itemList.addEventListener('click', removeItem);
clearBtn.addEventListener('click', clearItems);

checkUI(); // âœ… Cek UI saat page load
```
