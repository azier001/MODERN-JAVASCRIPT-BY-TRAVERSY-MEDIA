## ğŸ“– Apa itu Local Storage?

### Definisi

**Local Storage** adalah tempat penyimpanan data di browser yang bersifat permanen. Data tersimpan bahkan setelah browser ditutup!

### Analogi

Bayangkan Local Storage seperti **brankas di rumah**:
- ğŸ“¦ Anda bisa menyimpan barang (data) di dalamnya
- ğŸ”“ Anda bisa mengambil barang kapan saja
- ğŸ  Brankas itu tetap ada meskipun Anda pergi
- ğŸ“ Semua yang disimpan tercatat dengan jelas

### Bagaimana Kami Menggunakannya?

```javascript

localStorage.setItem('items', JSON.stringify(itemsFromStorage));

// Ambil item (BUKA brankas)
let itemsFromStorage = JSON.parse(localStorage.getItem('items'));

// Hapus item (KELUARKAN barang dari brankas)
localStorage.removeItem('items');
```

### Alur Ketika Edit Mode:

```
1. User klik item "Eggs" untuk edit
   â†“
2. Local Storage masih punya data lama: ["Juice", "Eggs", "Milk"]
   â†“
3. User ubah "Eggs" menjadi "Egg" dan klik Update
   â†“
4. HAPUS "Eggs" dari Local Storage
   Local Storage sekarang: ["Juice", "Milk"]
   â†“
5. TAMBAH "Egg" ke Local Storage
   Local Storage sekarang: ["Juice", "Milk", "Egg"]
```
