<a name="cara-membuat"></a>
## ğŸ”¨ Cara Membuat Constructor Function

Constructor function sebenarnya adalah **function biasa**, tapi dengan dua aturan konvensi:

1. **Huruf pertama nama function harus kapital** (misalnya `Rectangle`, bukan `rectangle`)
2. **Harus dipanggil dengan keyword `new`**

```javascript
// Membuat constructor function
function Rectangle() {
  // isi nanti...
}

// Menggunakan keyword `new` untuk membuat object baru
const rect1 = new Rectangle();

console.log(rect1); // Rectangle {} â€” object kosong terbentuk!
```

Kalau kita panggil **tanpa `new`**, hasilnya adalah `undefined` karena function-nya tidak mengembalikan apa-apa secara eksplisit.

```javascript
const rect1 = Rectangle(); // âŒ Tanpa new
console.log(rect1); // undefined
```

<a name="menambahkan-properties"></a>
## ğŸ·ï¸ Menambahkan Properties

Karena `this` sudah merujuk ke object baru, kita bisa menambahkan properties menggunakan `this.namaProperty`:

```javascript
function Rectangle(name, width, height) {
  this.name = name;     // property 'name' diisi dari argumen
  this.width = width;   // property 'width' diisi dari argumen
  this.height = height; // property 'height' diisi dari argumen
}

// Membuat object dengan data yang kita tentukan
const rect1 = new Rectangle('Rectangle 1', 10, 10);

console.log(rect1);
// Rectangle { name: 'Rectangle 1', width: 10, height: 10 }
```

Kita juga bisa akses property-nya langsung pakai dot notation, persis seperti object literal:

```javascript
console.log(rect1.name); // "Rectangle 1"
```

<a name="menambahkan-methods"></a>
## âš™ï¸ Menambahkan Methods

Selain properties biasa, kita bisa menambahkan **methods** (function di dalam object) dengan cara yang sama:

```javascript
function Rectangle(name, width, height) {
  this.name = name;
  this.width = width;
  this.height = height;

  // Method untuk menghitung luas
  this.area = function () {
    return this.height * this.width; // `this` merujuk ke object itu sendiri
  };
}

const rect1 = new Rectangle('Rectangle 1', 10, 10);

console.log(rect1.area()); // 100
```

> ğŸ’¡ Di dalam method, `this` merujuk ke object yang memanggil method tersebut â€” sama persis seperti di object literal.

<a name="banyak-instance"></a>
## ğŸ­ Membuat Banyak Instance

Inilah kekuatan utama constructor function â€” kita bisa membuat **sebanyak mungkin object** dari satu blueprint yang sama!

```javascript

  this.name = name;
  this.width = width;
  this.height = height;
  this.area = function () {
    return this.height * this.width;
  };
}

// Buat 3 object berbeda dari blueprint yang sama
const rect1 = new Rectangle('Rectangle 1', 10, 10);
const rect2 = new Rectangle('Rectangle 2', 20, 10);
const rect3 = new Rectangle('Rectangle 3', 30, 30);

// Akses property masing-masing
console.log(rect2.name, rect3.name);
// "Rectangle 2" "Rectangle 3"

// Panggil method masing-masing
console.log(rect2.area(), rect3.area());
// 200  900
```

Masing-masing object punya **data sendiri-sendiri**, tapi punya **struktur yang sama**. Keren kan? ğŸ‰
