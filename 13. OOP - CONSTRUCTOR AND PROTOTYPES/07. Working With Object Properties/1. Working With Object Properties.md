<a name="pengenalan"></a>
## ğŸ¯ Pengenalan

Di sini kita pakai **constructor function** untuk bikin objek `Rectangle`. Constructor ini punya properti `name`, `width`, `height`, dan method `area`.

```javascript
function Rectangle(name, width, height) {
  this.name = name;
  this.width = width;
  this.height = height;
  this.area = function () {
    return this.width * this.height;
  };
}
```

> ğŸ’¡ Sintaksnya mirip seperti object literal biasa â€” bedanya kita pakai `this` dan bisa bikin banyak objek dari satu template yang sama.

Ada dua cara untuk mengakses properti objek:

**1. Dot notation** â€” cara yang paling umum dipakai:

<a name="menambah-properti"></a>
## â• Menambah Properti

Kita bisa menambah properti baru ke objek kapan saja â€” bahkan setelah objek dibuat â€” cukup pakai dot notation dan assign nilainya.

**Menambah properti biasa:**

```javascript

// rect1 sekarang punya properti color: "red"
```

**Menambah method (fungsi) ke satu instance:**

```javascript

console.log(rect2.perimeter()); // 100
```

> âš ï¸ Perhatikan: method di atas pakai arrow function dan langsung sebut `rect2.width` â€” bukan `this.width`. Ini karena arrow function tidak punya `this` sendiri, jadi kalau kita tulis `this` di sini, yang dimaksud adalah `window` (bukan objeknya). Makanya kita langsung referensikan nama variabelnya.

> ğŸ“Œ Menambah method ke satu instance saja jarang dilakukan dalam praktik nyata. Kalau ingin semua objek punya method tersebut, lebih baik tambahkan langsung ke constructor atau ke prototype.

## ğŸ—‚ï¸ Object.keys, values, entries

JavaScript punya tiga method bawaan yang berguna untuk mengambil data dari objek dalam bentuk array.

**`Object.keys()`** â€” mengambil semua nama properti (termasuk method):

```javascript

```

**`Object.values()`** â€” mengambil semua nilai properti:

```javascript

// ["Rectangle 2", 20, 30, Æ’]
```

**`Object.entries()`** â€” mengambil pasangan key-value, masing-masing dalam bentuk array:

```javascript

// [["name", "Rectangle 1"], ["width", 20], ["height", 20], ...]
```

<a name="looping"></a>
## ğŸ”„ Looping dengan for...of

Kita bisa loop lewat `Object.entries()` menggunakan `for...of` dan destructuring untuk dapat key & value sekaligus:

```javascript
for (let [key, value] of Object.entries(rect1)) {
  console.log(`${key} - ${value}`);
}
// name - Rectangle 1
// width - 20
// height - 20
// area - function...
// color - red
```

**Kalau hanya mau properti, tanpa method**, tambahkan pengecekan tipe:

```javascript
for (let [key, value] of Object.entries(rect1)) {
  if (typeof value !== 'function') {
    console.log(`${key} - ${value}`);
  }
}
// name - Rectangle 1
// width - 20
// height - 20
// color - red
```

> ğŸ’¡ `typeof` fungsi akan mengembalikan string `"function"`, jadi kita bisa filter mereka keluar dengan mudah.
