<a name="membuat-object-literal-prototype"></a>
## ğŸ§± Membuat Object Literal Prototype

Langkah pertama adalah membuat sebuah object biasa yang berisi semua method yang ingin kita jadikan prototype.

```javascript
const rectanglePrototypes = {
  // Menghitung luas persegi panjang
  area: function () {
    return this.width * this.height;
  },

  // Menghitung keliling persegi panjang
  perimeter: function () {
    return 2 * (this.width + this.height);
  },

  // Mengecek apakah bentuknya persegi (bukan persegi panjang)
  isSquare: function () {
    return this.height === this.width;
  },
};
```

> ğŸ“ **Catatan:** Object ini sendiri belum melakukan apa-apa. Ini hanya "tempat penyimpanan" untuk method-method kita. Selanjutnya kita perlu menghubungkannya ke object baru.

<a name="fungsi-createrectangle"></a>
## ğŸ—ï¸ Fungsi `createRectangle` dengan `Object.create`

Sekarang kita buat sebuah fungsi yang tugasnya membuat object rectangle baru. Di dalam fungsi ini kita pakai `Object.create`.

```javascript
function createRectangle(height, width) {
  return Object.create(rectanglePrototypes, {
    height: {
      value: height,   // nilai diambil dari parameter fungsi
    },
    width: {
      value: width,    // nilai diambil dari parameter fungsi
    },
  });
}
```

### ğŸ” Penjelasan `Object.create`

`Object.create` menerima **dua argumen**:

1. **Argumen pertama** â†’ object yang akan dijadikan prototype (yaitu `rectanglePrototypes` yang kita buat tadi)
2. **Argumen kedua** â†’ object berisi **properti-properti** yang ingin kita tambahkan ke object baru, dengan format:
   ```javascript
   {
     namaProperti: { value: nilainya }
   }
   ```

Jadi `Object.create` ini tugasnya: *"Buat object baru, gunakan `rectanglePrototypes` sebagai prototypenya, dan tambahkan properti `height` serta `width`."*

<a name="ringkasan"></a>
## ğŸ“Œ Ringkasan

Cara alternatif membuat object dengan prototype menggunakan `Object.create`:

1. **Buat object literal** berisi semua method prototype â†’ `rectanglePrototypes`
2. **Buat fungsi factory** yang menggunakan `Object.create(prototypeObject, propertiesObject)`
3. **Panggil fungsi** tersebut setiap kali mau membuat object baru
4. **Gunakan method** langsung dari instance tanpa perlu `.prototype`

```javascript
// Kode lengkap
const rectanglePrototypes = {
  area: function () { return this.width * this.height; },
  perimeter: function () { return 2 * (this.width + this.height); },
  isSquare: function () { return this.height === this.width; },
};

function createRectangle(height, width) {
  return Object.create(rectanglePrototypes, {
    height: { value: height },
    width: { value: width },
  });
}

const rect = createRectangle(10, 20);
console.log(rect.area());     // 200
console.log(rect.isSquare()); // false
```
