<a name="constructor-function"></a>
## ğŸ—ï¸ Membuat Constructor Function

Constructor function adalah "cetakan" untuk membuat object. Setiap kali kita tulis `new Player('Bob')`, JavaScript akan menjalankan function ini dan menghasilkan object baru.

**Langkah 1 â€” Buat constructor dasarnya dulu:**

```js
function Player(name) {
  this.name = name;
}
```

ğŸ’¡ **Apa yang terjadi di sini?**
- `function Player` â€” nama diawali huruf kapital, itu konvensi untuk constructor function
- `this.name = name` â€” properti `name` diisi dari argumen yang kita masukkan

**Langkah 2 â€” Tambahkan properti default:**

```js
function Player(name) {
  this.name = name;
  this.lvl = 1;     // semua player mulai dari level 1
  this.points = 0;  // dan punya 0 poin di awal
}
```

ğŸ’¡ **Apa yang terjadi di sini?**
- `this.lvl = 1` â€” setiap player baru otomatis mulai di level 1
- `this.points = 0` â€” poin awal selalu 0, tidak perlu diisi manual

<a name="gainxp"></a>
## âš”ï¸ Prototype Method: `gainXp`

Method ini menambah XP ke player. Kalau poin sudah mencapai 10 atau lebih, player **naik level** dan poin dikurangi 10.

Kenapa ditaruh di `prototype`? Supaya semua object `Player` berbagi satu fungsi yang sama â€” lebih hemat memori daripada bikin function baru di tiap object.

**Langkah 1 â€” Tambahkan poin dulu:**

```js
Player.prototype.gainXp = function(xp) {
  this.points += xp; // tambahkan, bukan ganti!
};
```

ğŸ’¡ `+=` dipakai agar XP *ditambahkan* ke poin yang sudah ada, bukan mengganti nilainya.

**Langkah 2 â€” Tambahkan logika naik level:**

```js
Player.prototype.gainXp = function(xp) {
  this.points += xp;

  if (this.points >= 10) {
    this.lvl++;          // naik 1 level
    this.points -= 10;   // kurangi 10 poin
  }
};
```

ğŸ’¡ **Apa yang terjadi di sini?**
- `if (this.points >= 10)` â€” cek apakah poin sudah cukup untuk naik level
- `this.lvl++` â€” increment level sebanyak 1
- `this.points -= 10` â€” kurangi poin, bukan reset ke 0. Sisa poin tetap terbawa ke level berikutnya!

<a name="describe"></a>
## ğŸ“‹ Prototype Method: `describe`

Method ini mengembalikan string yang berisi statistik player: nama, level, dan poin.

```js
Player.prototype.describe = function() {
  return `${this.name} is level ${this.lvl} with ${this.points} experience points`;
};
```

ğŸ’¡ **Apa yang terjadi di sini?**
- Menggunakan **template literal** (backtick) agar variabel bisa disisipkan langsung ke dalam string
- `this.name`, `this.lvl`, `this.points` â€” mengakses properti object yang memanggil method ini
- Method ini `return` string, jadi perlu `console.log()` di luar untuk menampilkannya

ğŸ’¡ **Apa yang terjadi di sini?**

- `describe()` dipanggil di akhir untuk melihat stats final setiap player
