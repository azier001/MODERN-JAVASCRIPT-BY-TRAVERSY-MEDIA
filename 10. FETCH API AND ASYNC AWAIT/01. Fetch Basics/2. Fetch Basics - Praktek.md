<a name="langkah-1-fetching-file-json"></a>
## ğŸ¬ Langkah 1: Fetching File JSON

### ğŸ”¹ Kode Dasar (Cara Panjang)

Buka `script.js`, tulis kode ini:

```javascript
// Fetching a JSON file
fetch('./movies.json')
  .then(function(response) {
    console.log(response);
    return response.json();
  })
  .then(function(data) {
    console.log(data);
  });
```

**Penjelasan:**
- `fetch('./movies.json')` â†’ Membuat request ke file movies.json
- `.then()` â†’ Karena fetch mengembalikan Promise, kita pakai `.then()`
- `response` â†’ Object yang berisi informasi tentang response
- `response.json()` â†’ Method untuk mengubah response menjadi JSON
- `data` â†’ Data JSON yang sudah siap digunakan

### ğŸ”¹ Kode Pendek (Menggunakan Arrow Function)

Kode yang sama, tapi lebih ringkas:

```javascript

fetch('./movies.json')
  .then((response) => response.json())
  .then((data) => console.log(data));
```

**âœ… Coba jalankan:**
1. Buka `index.html` di browser
2. Buka Console (F12 â†’ Console)

## ğŸ’¡ Tips Penting

### âœ… GET Request (Default)

Semua fetch yang kita lakukan di atas adalah **GET request**:

```javascript

```

### ğŸ” Authentikasi API

Beberapa API memerlukan **API Key** atau **Token**:

```javascript

})
```

GitHub API tidak perlu auth untuk beberapa request pertama, makanya kita bisa langsung pakai!

### ğŸš€ Axios Alternative

```javascript

  .then((data) => console.log(data)) // Langsung dapat data!
```

Tapi kita pakai Fetch karena sudah built-in di browser! ğŸ‰

### âš¡ Async/Await

Ada cara lain menulis Fetch menggunakan `async/await` (akan dipelajari nanti):

```javascript

  const response = await fetch('https://api.github.com/users/bradtraversy');
  const data = await response.json();
  console.log(data);
```
