## ğŸ“– Apa itu Try Catch?

**Try Catch** adalah struktur kode di JavaScript yang memungkinkan kita untuk:
- âœ… **Mencoba** (`try`) menjalankan kode yang mungkin menghasilkan error
- âŒ **Menangkap** (`catch`) error jika terjadi
- ğŸ”„ Menjalankan kode alternatif saat error terjadi

**Syntax dasar:**
```javascript

} catch (error) {
  // Kode yang dijalankan jika ada error
}
```

### ğŸ“Œ Penjelasan Baris per Baris:

#### ğŸ“Œ Penjelasan Baris per Baris:

```javascript

```
- `num2 === 0` mengecek apakah pembagi adalah nol
- Pembagian dengan 0 tidak terdefinisi dalam matematika
- Error dilempar dengan pesan spesifik

**Baris 11-13: Return Hasil**

- Jika lolos semua validasi, baru lakukan pembagian
- Operator `/` untuk pembagian

**Baris 16-21: Penggunaan Fungsi**
