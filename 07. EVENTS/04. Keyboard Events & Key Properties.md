## ğŸ“ Kesimpulan

### ğŸ“ Ringkasan Yang Sudah Dipelajari:

| Event | Kapan Terjadi | Kegunaan |
|-------|---------------|----------|
| **keypress** | Saat tombol ditekan (1x) | Deteksi ketukan biasa |
| **keyup** | Saat tombol dilepas | Deteksi "selesai mengetik" |
| **keydown** | Saat tombol ditahan | Game, gerakan kontinu |

### ğŸ”‘ Cara Mengetahui Tombol Yang Ditekan:

| Properti | Format | Contoh |
|----------|--------|--------|
| `e.key` | Nama tombol | `Enter`, `A`, `1` |
| `e.keyCode` | Nomor kode | `13`, `65`, `49` |
| `e.code` | Key+Nama / Digit+Angka | `KeyA`, `Digit1` |

### âš¡ Properti Tambahan:

- `e.repeat` â†’ Apakah tombol ditahan? (true/false)
- `e.shiftKey` â†’ Apakah Shift ditekan? (true/false)
- `e.ctrlKey` â†’ Apakah Ctrl ditekan? (true/false)
- `e.altKey` â†’ Apakah Alt ditekan? (true/false)

## ğŸš€ Tips Penggunaan:

1. **Untuk deteksi tombol biasa:** Gunakan `e.key` (paling mudah!)
2. **Untuk game dengan tombol ditahan:** Gunakan `keydown` + `e.repeat`
3. **Untuk shortcut keyboard:** Gabungkan modifier keys (`e.shiftKey && e.key === 'S'`)
4. **Untuk browser lama:** Gunakan `e.keyCode`
