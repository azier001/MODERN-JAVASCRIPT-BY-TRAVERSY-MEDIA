## ğŸ›‘ Menghentikan Event Bubbling

Kadang kita **tidak ingin** event bubbling terjadi. Untuk itu, kita bisa menggunakan method `stopPropagation()`.

### ğŸ“ Kapan Menggunakan `stopPropagation()`?

Gunakan ketika:
- âŒ Event bubbling menyebabkan bug atau perilaku tidak diinginkan
- âŒ Kamu hanya ingin event terjadi di elemen yang diklik saja
- âŒ Parent element memiliki event yang sama dan tidak boleh terpicu

### ğŸ’¡ Cara Menggunakan:

```javascript

button.addEventListener('click', (e) => {
  alert('Button was clicked');
  
  // ğŸ›‘ STOP! Jangan bubble up ke parent!
  e.stopPropagation();
});
```

**Penjelasan**:
- `(e)` â†’ Parameter event object
- `e.stopPropagation()` â†’ Method untuk menghentikan event bubbling

**Hasil Setelah Menggunakan `stopPropagation()`**:
- âœ… Ketika BUTTON diklik â†’ **HANYA** muncul alert "Button was clicked"
- âœ… Event **TIDAK** bubble up ke DIV, FORM, atau BODY
- âœ… Alert lainnya **TIDAK** muncul!

### âš ï¸ Peringatan Penting:

> **JANGAN** menggunakan `stopPropagation()` tanpa alasan yang jelas! Hanya gunakan ketika kamu benar-benar membutuhkannya, misalnya ketika parent dan child memiliki event listener yang sama dan kamu ingin mencegah konflik.
