## ğŸ¨ Sintaks Dasar Filter

### Format 1: Function Declaration (Panjang)

```javascript

  return kondisi; // true atau false

```

**Penjelasan:**
- `array.filter()` â†’ method filter pada array
- `function(element)` â†’ function yang diterima, `element` = item saat ini
- `return kondisi` â†’ harus return `true` atau `false`
  - `true` = elemen masuk array hasil
  - `false` = elemen ditolak

### Format 2: Arrow Function (Singkat) âœ¨

```javascript

```

**Catatan:**
- Jika hanya 1 parameter, **tanda kurung bisa dihilangkan**: `element =>` bukan `(element) =>`
- Jika hanya 1 baris, **tanda kurung kurawal dan return bisa dihilangkan**

### Contoh Sintaks:

```javascript

  return num % 2 === 0;
});

// Sedang
const genap2 = numbers.filter((num) => {
  return num % 2 === 0;
});

// Paling Singkat â­
const genap3 = numbers.filter(num => num % 2 === 0);

// Semuanya menghasilkan output yang SAMA!
```
