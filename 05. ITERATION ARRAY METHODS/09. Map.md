## ğŸ“– Apa itu Array `.map()`?

### Definisi Sederhana

`.map()` adalah **method array** yang memungkinkan kita **mengubah setiap elemen** dalam array menjadi elemen baru, kemudian mengembalikan array baru yang berisi hasil transformasi tersebut.

### Karakteristik Penting âœ¨

| Fitur | Penjelasan |
|-------|-----------|
| **Membuat Array Baru** | `.map()` selalu membuat array baru, tidak mengubah array asli |
| **Iterasi Otomatis** | Otomatis loop melalui setiap elemen |
| **Fleksibel** | Bisa mengembalikan apa saja: angka, string, objek, dll |
| **Urutan Terjaga** | Elemen baru mempertahankan urutan yang sama |

> ğŸ’¡ **Konsep Kunci:** Setiap elemen diproses satu per satu, dan hasil pengembalian dari fungsi akan menjadi elemen baru di array hasil

## ğŸ”„ Perbedaan Map vs Filter

### `.filter()` ğŸ¯
- Menyeleksi elemen berdasarkan kondisi `true` atau `false`
- Hanya elemen yang mengembalikan `true` yang masuk ke array baru
- **Tujuan:** Memperkecil array

### `.map()` ğŸ”€
- Mengubah **setiap elemen** menjadi sesuatu yang baru
- Fungsi dapat mengembalikan **apa saja** (bukan hanya true/false)
- **Tujuan:** Transformasi setiap elemen

### Ilustrasi Perbedaan

```

         â†“ Berkurang

Map:    [1, 2, 3, 4, 5] â†’ Kalikan 2 â†’ [2, 4, 6, 8, 10]
         â†“ Jumlah tetap sama, nilai berubah
```

## ğŸ” Menggunakan forEach Sebagai Alternatif

### Apa itu forEach?

`.forEach()` adalah cara lain untuk loop, tetapi **tidak membuat array baru**. Kita harus membuat array kosong terlebih dahulu dan `.push()` hasil ke dalamnya.

### Perbandingan Map vs forEach

**Dengan `.map()` (Direkomendasikan):**
```javascript
const numbers = [1, 2, 3, 4, 5];
const doubledNumbers = numbers.map((number) => number * 2);

console.log(doubledNumbers);
// Hasil: [2, 4, 6, 8, 10]
```

**Dengan `.forEach()` (Lebih panjang):**
```javascript
const numbers = [1, 2, 3, 4, 5];
const doubledNumbers = [];

numbers.forEach((number) => {
  doubledNumbers.push(number * 2);
});

console.log(doubledNumbers);
// Hasil: [2, 4, 6, 8, 10]
```

### Kesimpulan ğŸ’¡

| Method | Kelebihan | Kekurangan |
|--------|-----------|-----------|
| `.map()` | âœ… Singkat, bersih, langsung return array | âŒ Selalu membuat array baru |
| `.forEach()` | âœ… Fleksibel untuk operasi samping | âŒ Lebih panjang, manual push |

> **Rekomendasi:** Gunakan `.map()` ketika ingin mengubah array â†’ menghasilkan array baru! ğŸ¯

## â›“ï¸ Chaining Multiple Map Methods

### Apa itu Chaining?

**Chaining** = Menggabungkan beberapa method sekaligus. Hasil dari method pertama langsung masuk ke method berikutnya.

### Contoh: Square Root dan Kemudian Kalikan 2

**Tujuan:** 
1. Ambil akar kuadrat dari setiap angka
2. Kemudian kalikan hasilnya dengan 2

#### Cara Singkat (Recommended):

```javascript
const numbers = [1, 4, 9, 16, 25];

const squareAndDouble = numbers
  .map((number) => Math.sqrt(number))
  .map((sqrt) => sqrt * 2);

console.log(squareAndDouble);
// Hasil: [2, 4, 6, 8, 10]
```

**Cara Kerjanya:**
```
Step 1 - Map pertama (Square Root):
  [1, 4, 9, 16, 25] â†’ [1, 2, 3, 4, 5]
                       â†“
Step 2 - Map kedua (Kalikan 2):
  [1, 2, 3, 4, 5] â†’ [2, 4, 6, 8, 10] âœ¨
```

#### Cara Panjang (Untuk Pemahaman):

```javascript
const squareAndDouble2 = numbers
  .map(function (number) {
    return Math.sqrt(number);
  })
  .map(function (sqrt) {
    return sqrt * 2;
  });

console.log(squareAndDouble2);
// Hasil sama: [2, 4, 6, 8, 10]
```

---

### Contoh dengan 3 Map Methods:

Kita bisa terus chain sesuai kebutuhan!

```javascript
const squareTripleTransform = numbers
  .map((number) => Math.sqrt(number))     // Step 1: Akar kuadrat
  .map((sqrt) => sqrt * 2)                 // Step 2: Kalikan 2
  .map((doubled) => doubled * 3);          // Step 3: Kalikan 3

console.log(squareTripleTransform);
// Untuk [1, 4, 9, 16, 25]:
// Step 1: [1, 2, 3, 4, 5]
// Step 2: [2, 4, 6, 8, 10]
// Step 3: [6, 12, 18, 24, 30]
```

> âš ï¸ **Penting:** Setiap `.map()` hanya bisa mengakses hasil dari `.map()` sebelumnya, bukan sebelum-sebelumnya!
