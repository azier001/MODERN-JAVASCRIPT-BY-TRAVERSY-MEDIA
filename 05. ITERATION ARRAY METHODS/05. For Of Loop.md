## ğŸ“– Apa itu For Of Loop?

**For Of Loop** adalah cara yang lebih baru dan lebih mudah dibaca untuk melakukan perulangan melalui **objek yang dapat diiterasi** (iterable objects). 

Objek-objek yang dapat diiterasi meliputi:
- ğŸ“š **Array** (kumpulan data)
- ğŸ“ **String** (teks)
- ğŸ—ºï¸ **Map** (pasangan kunci-nilai)
- ğŸ“¦ **Set** (kumpulan nilai unik)

Bayangkan kamu punya daftar belanja dan ingin membacanya satu per satu. For Of Loop adalah cara yang paling mudah untuk melakukannya!

<a name="mengapa-menggunakan-for-of-loop"></a>
## âœ¨ Mengapa Menggunakan For Of Loop?

Mari kita bandingkan dengan cara lama (Standard For Loop) untuk melihat perbedaannya:

### âŒ Cara Lama (Standard For Loop)

```javascript
const items = ['Book', 'Table', 'Chair', 'Kite'];

for (let i = 0; i < items.length; i++) {
  console.log(items[i]); // Akses melalui index
}
```

**Masalah:** 
- Harus membuat variabel `i` terlebih dahulu
- Harus menghitung panjang array dengan `.length`
- Harus menggunakan index `[i]` untuk akses data
- Kode terlihat rumit dan sulit dibaca

### âœ… Cara Baru (For Of Loop)

```javascript
const items = ['Book', 'Table', 'Chair', 'Kite'];

for (const item of items) {
  console.log(item); // Langsung akses datanya!
}
```

**Keuntungan:**
- âœ¨ Kode lebih singkat dan rapi
- ğŸ¯ Lebih mudah dipahami
- ğŸš€ Lebih modern dan profesional
- ğŸ’ª Tidak perlu khawatir tentang index

## ğŸ“š Looping Melalui Array

Array adalah kumpulan data yang paling sering kita gunakan. Mari kita lihat bagaimana menggunakan For Of Loop dengan array.

### Contoh 1: Array Sederhana

```javascript
// Buat array dengan beberapa item
const items = ['Book', 'Table', 'Chair', 'Kite'];

// Looping melalui setiap item
for (const item of items) {
  console.log(item);
}
```

**Output:**
```
Book
Table
Chair
Kite
```

ğŸ’¡ **Penjelasan:** Setiap kali loop berjalan, variabel `item` akan berisi satu item dari array, dimulai dari `'Book'` hingga `'Kite'`.

---

### Contoh 2: Array dengan Objek

Sering kali kita punya array yang berisi objek (data yang lebih kompleks). For Of Loop juga berfungsi sempurna untuk ini!

```javascript
// Buat array yang berisi objek-objek
const users = [
  { name: 'Brad' },
  { name: 'Kate' },
  { name: 'Steve' }
];

// Looping melalui setiap user
for (const user of users) {
  console.log(user.name); // Akses properti 'name'
}
```

**Output:**
```
Brad
Kate
Steve
```

ğŸ’¡ **Penjelasan:** Setiap item dalam loop adalah objek. Kita menggunakan `user.name` untuk mengakses properti `name` dari setiap objek.

---

### Contoh 3: Array dengan Berbagai Tipe Data

Array juga bisa berisi berbagai tipe data:

```javascript

for (const item of mixedArray) {
  console.log(item);
}
```

**Output:**

<a name="looping-melalui-map"></a>
## ğŸ—ºï¸ Looping Melalui Map

Map adalah struktur data yang menyimpan pasangan **kunci-nilai** (key-value pairs). Ini berguna ketika kamu ingin menyimpan data dengan label yang jelas.

### Penjelasan Map

Bayangkan kamu punya data user:
- Kunci: `'name'` â†’ Nilai: `'John'`
- Kunci: `'age'` â†’ Nilai: `30`

Inilah yang dilakukan Map!

### Contoh: Membuat dan Mengiterasi Map

```javascript
// Buat sebuah Map baru
const map = new Map();

// Tambahkan kunci-nilai pairs
map.set('name', 'John');
map.set('age', 30);

// Looping melalui Map
for (const [key, value] of map) {
  console.log(key, value);
}
```

**Output:**
```
name John
age 30
```

### ğŸ¯ Perhatian: Sintaks Destructuring

```javascript
for (const [key, value] of map) {
  // [key, value] adalah destructuring
  // Ini membuat kita bisa langsung akses key dan value
}
```

Kurung siku `[ ]` di sini bukan array biasa, tapi **destructuring assignment**. Ini cara cepat untuk mengambil kunci dan nilai dari setiap item Map.

### Contoh Lebih Kompleks: Data Pengguna

```javascript
// Buat Map dengan data pengguna
const userInfo = new Map();
userInfo.set('name', 'Alice');
userInfo.set('email', 'alice@example.com');
userInfo.set('country', 'Indonesia');

// Tampilkan semua informasi
for (const [key, value] of userInfo) {
  console.log(`${key.toUpperCase()}: ${value}`);
}
```

**Output:**
```
NAME: Alice
EMAIL: alice@example.com
COUNTRY: Indonesia
```

### ğŸ’¡ Tips Penting untuk Pemula

1. **Gunakan `const` untuk variabel perulangan**
   ```javascript

   // âŒ Hindari menggunakan `let` jika tidak perlu
   ```

2. **Jangan menggunakan `for...in` untuk array**
   ```javascript

   // âœ… Benar untuk array

   ```

3. **Jangan lupa `:` di akhir nama variabel di Map**
   ```javascript
   // âœ… Benar
   for (const [key, value] of map) { }
   
   // âŒ Lupa kurung kotak
   for (const key, value of map) { }
   ```

4. **For Of Loop hanya untuk objek yang dapat diiterasi**
   ```javascript
   // âœ… Bisa: array, string, map, set
   // âŒ Tidak bisa: objek literal biasa, number
   ```
