## ğŸ“– Apa itu For...In Loop?

**For...In Loop** adalah sebuah cara untuk mengulang melalui **semua properti (key/kunci)** dari sebuah object atau array.

### ğŸ” Sintaks Dasar

```javascript

  // Kode yang akan diulang
}
```

### ğŸ“ Penjelasan:

| Bagian | Penjelasan |
|--------|-----------|
| `for` | Kata kunci untuk membuat loop |
| `const variabel` | Variabel yang akan menyimpan setiap key |
| `in` | Kata kunci yang berarti "dalam" atau "di dalam" |
| `object` | Object atau array yang ingin kita loop |
| `{ ... }` | Blok kode yang akan diulang setiap iterasi |

## âš ï¸ Perbedaan Penting: Object vs Array

| Aspek | Object | Array |
|-------|--------|-------|
| **Struktur** | Pasangan key-value | List urutan item |
| **Key dalam For...In** | Nama properti (string) | Index/nomor (0, 1, 2...) |
| **Contoh Key** | `'color1'`, `'name'` | `0`, `1`, `2` |
| **Akses Value** | `obj['color1']` | `arr[0]` |

---

### ğŸ“ Contoh Perbandingan

**Object:**
```javascript

  brand: 'Toyota',
  color: 'blue',
  year: 2020
};

}
// Output:
// brand: Toyota
// color: blue
// year: 2020
```

**Array:**
```javascript

for (const key in fruits) {
  console.log(`Index ${key}: ${fruits[key]}`);
}
// Output:
// Index 0: apple
// Index 1: banana
// Index 2: orange
```

## âœ… Kesimpulan

### ğŸ¯ Poin-Poin Penting:

1. **For...In Loop** digunakan untuk **mengulang melalui keys** dari object atau array
2. Pada **object**, key adalah nama properti (misal: `'color1'`, `'nama'`)
3. Pada **array**, key adalah index/nomor (misal: `0`, `1`, `2`)
4. Untuk mendapatkan **nilai**, gunakan **bracket notation** `[key]`
5. For...In Loop paling sering digunakan untuk **object literal**

### ğŸš€ Langkah Berikutnya:

Setelah memahami For...In Loop, pelajari juga:
- âœ¨ **forEach()** - cara modern untuk loop array
- âœ¨ **map()** - transform setiap item di array
- âœ¨ **filter()** - pilih item tertentu dari array
