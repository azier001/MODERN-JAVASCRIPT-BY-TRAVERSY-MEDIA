### ğŸ” Loop (Perulangan)

Loop adalah cara untuk mengulang suatu aksi berkali-kali. Ada dua jenis yang akan kita gunakan:

**1. For Loop**
```javascript

  console.log(i); // Akan print 1 sampai 10
}
```

**2. While Loop**
```javascript
let i = 1;
while (i <= 10) {
  console.log(i); // Akan print 1 sampai 10
  i++;
}
```

<a name="modulus"></a>
### â— Modulus Operator (%)

**Modulus** adalah operator yang memberikan **sisa hasil bagi**:
