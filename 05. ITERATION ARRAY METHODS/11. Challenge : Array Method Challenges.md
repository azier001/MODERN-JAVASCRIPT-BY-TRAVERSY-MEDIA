### ðŸ’¡ Konsep Dasar: Callback Function


## ðŸ“– Method Filter Explained

### â“ Apa itu `.filter()`?

Method `.filter()` digunakan untuk **menyaring** atau **memilih** elemen tertentu dari array berdasarkan kondisi yang kita tentukan.

### ðŸ“Š Cara Kerja Filter

```
Array Awal: [1, 2, 3, 4, 5]
                â†“
        Kondisi: x > 3 ?
                â†“
Hasil Akhir: [4, 5]
```

### ðŸ’» Syntax Dasar

```javascript
const hasil = array.filter((element) => {
  return kondisi; // true atau false
});
```

### ðŸ” Penjelasan:
- `array` - Array yang ingin kita filter
- `element` - Setiap item dalam array
- `kondisi` - Pernyataan true/false yang menentukan item dipilih atau tidak
- Jika kondisi **true**, item dimasukkan ke array baru
- Jika kondisi **false**, item ditolak

### ðŸ“ Contoh Real-World

```javascript

// Pilih hanya yang berusia di atas 18 tahun
const dewasa = umur.filter((age) => age >= 18);

console.log(dewasa); // [18, 25, 30]
```

## ðŸ“– Method Map Explained

### â“ Apa itu `.map()`?

Method `.map()` digunakan untuk **mengubah** atau **mentransformasi** setiap elemen dalam array menjadi bentuk baru, kemudian membuat array baru dengan hasil transformasi tersebut.

### ðŸ“Š Cara Kerja Map

```
                    â†“
        Transformasi: uppercase
                    â†“
```

### ðŸ’» Syntax Dasar

### ðŸ” Penjelasan:
- Array awal **tidak berubah** (immutable)
- Setiap elemen ditransformasi ke bentuk baru
- Array baru dibuat dengan panjang yang sama
- `return` apa yang ingin jadi elemen baru

### ðŸ“ Contoh Real-World


## ðŸ“– Method Reduce Explained

### â“ Apa itu `.reduce()`?

Method `.reduce()` digunakan untuk **mengumpulkan** atau **mengakumulasi** semua elemen dalam array menjadi **satu nilai tunggal**.

### ðŸ“Š Cara Kerja Reduce

```
Array: [1, 2, 3, 4]
         â†“
Akumulasi: 1 + 2 = 3
           3 + 3 = 6
           6 + 4 = 10
         â†“
Hasil: 10
```

### ðŸ’» Syntax Dasar

```javascript
const hasil = array.reduce((akumulator, elemen) => {
  return akumulator + elemen;
}, nilaiAwal);
```

### ðŸ” Penjelasan:
- `akumulator` - Nilai yang terakumulasi (dimulai dari `nilaiAwal`)
- `elemen` - Elemen saat ini yang sedang diproses
- `nilaiAwal` - Nilai awal untuk akumulator (biasanya 0 untuk penjumlahan)
- Setiap iterasi, akumulator diupdate dengan hasil kalkulasi

### ðŸ“ Contoh Real-World

```javascript
const nilai = [70, 80, 90];

// Hitung rata-rata
const total = nilai.reduce((sum, n) => sum + n, 0);
const rata = total / nilai.length;

console.log(total); // 240
console.log(rata);  // 80
```

- Setelah filter, kita punya 3 orang

## ðŸŽ¯ Challenge 2: Filter & Reduce

### ðŸ“‹ Deskripsi Challenge

Kamu diberikan array berisi angka positif dan negatif.

**Tugas:**
Hitung **TOTAL dari semua angka positif saja** (abaikan angka negatif).

### ðŸ“Š Data Input

```javascript
const numbers = [2, -30, 50, 20, -12, -9, 7];
```

### âœ¨ Expected Result

```javascript
console.log(positiveSum); // 79
// Penjelasan: 2 + 50 + 20 + 7 = 79
```

### ðŸ”‘ Langkah-Langkah Menyelesaikan

#### Langkah 1: Filter angka positif

Gunakan `.filter()` untuk memilih hanya angka yang **> 0**

```javascript
const positiveNumbers = numbers.filter((number) => {
  return number > 0;
});

console.log(positiveNumbers); // [2, 50, 20, 7]
```

**Penjelasan:**
- Iterasi setiap `number` dalam array
- Jika lebih besar dari 0, masukkan ke array baru
- Angka negatif (-30, -12, -9) dibuang

#### Langkah 2: Jumlahkan dengan Reduce

Gunakan `.reduce()` untuk menambahkan semua angka dalam array

```javascript
const positiveSum = numbers
  .filter((number) => number > 0)
  .reduce((accumulator, number) => {
    return accumulator + number;
  }, 0);

console.log(positiveSum); // 79
```

**Penjelasan Reduce:**
```
Nilai Awal (0):        acc = 0
Iterasi 1 (2):         acc = 0 + 2 = 2
Iterasi 2 (50):        acc = 2 + 50 = 52
Iterasi 3 (20):        acc = 52 + 20 = 72
Iterasi 4 (7):         acc = 72 + 7 = 79
Hasil Akhir:           79
```

### ðŸ’¡ Versi Singkat (Arrow Function)

```javascript
const positiveSum = numbers
  .filter((number) => number > 0)
  .reduce((acc, cur) => acc + cur, 0);

console.log(positiveSum); // 79
```

### ðŸ“Œ Solusi Final

```javascript
const numbers = [2, -30, 50, 20, -12, -9, 7];

const positiveSum = numbers
  .filter((number) => number > 0)
  .reduce((acc, cur) => acc + cur, 0);

console.log(positiveSum); // 79
```

## ðŸŽ¯ Challenge 3: Map untuk Transformasi

### ðŸ“‹ Deskripsi Challenge

Kamu diberikan array berisi kata-kata dalam huruf **lowercase** (kecil).

**Tugas:**
Buat array baru bernama `capitalizedWords` dimana **huruf pertama dari setiap kata di-uppercase** (besar), sementara huruf sisanya tetap lowercase.

### ðŸ“Š Data Input


### âœ¨ Expected Result

### ðŸ”‘ Langkah-Langkah Menyelesaikan

#### Langkah 1: Gunakan Map untuk Transform

#### Langkah 2: Ambil Huruf Pertama & Uppercase-kan

Untuk mendapatkan huruf pertama, gunakan **index** seperti array biasa

Untuk mengambil karakter mulai dari index ke-1 sampai akhir, gunakan `.slice()`

```javascript

// Contoh: 'coder'
// word.slice(1) = 'oder'
```

Gunakan **concatenation** atau **template string** untuk menggabungkan

```javascript

  return word[0].toUpperCase() + word.slice(1);

// ['Coder', 'Programmer', 'Developer']
```

**Breakdown per iterasi:**
```
Iterasi 1: 'coder'
  â†’ word[0].toUpperCase() = 'C'
  â†’ word.slice(1) = 'oder'
  â†’ Hasil: 'C' + 'oder' = 'Coder'

Iterasi 2: 'programmer'
  â†’ word[0].toUpperCase() = 'P'
  â†’ word.slice(1) = 'rogrammer'
  â†’ Hasil: 'P' + 'rogrammer' = 'Programmer'

Iterasi 3: 'developer'
  â†’ word[0].toUpperCase() = 'D'
  â†’ word.slice(1) = 'eveloper'
  â†’ Hasil: 'D' + 'eveloper' = 'Developer'
```

### ðŸ’¡ Versi dengan Template String

```javascript

  (word) => `${word[0].toUpperCase()}${word.slice(1)}`
);

console.log(capitalizedWords);
// ['Coder', 'Programmer', 'Developer']
```

### ðŸ“Œ Solusi Final

```javascript

  (word) => `${word[0].toUpperCase()}${word.slice(1)}`
```

<a name="tips-dan-trik"></a>
## ðŸ’¡ Tips dan Trik

### 1ï¸âƒ£ Chaining adalah Kuat

Kamu bisa menggabungkan multiple methods sekaligus:

```javascript
// âœ… BAIK - Chaining

  .filter(condition1)
  .map(transform1)
  .filter(condition2)
  .reduce(accumulator, 0);

// âŒ KURANG - Berulang-ulang
const step1 = array.filter(condition1);
const step2 = step1.map(transform1);
const step3 = step2.filter(condition2);
const result = step3.reduce(accumulator, 0);
```

### 2ï¸âƒ£ Gunakan Arrow Function untuk Singkat

```javascript
// âœ… SINGKAT
.filter(x => x > 0)

// âŒ PANJANG
.filter((x) => {
  return x > 0;
})
```

### 3ï¸âƒ£ Immutability - Jangan Ubah Data Original

```javascript
// âœ… BAIK - Membuat array baru
const newArray = oldArray.map(x => x * 2);

// âŒ KURANG - Mengubah array original
oldArray.forEach((x, i) => {
  oldArray[i] = x * 2;
});
```

### 4ï¸âƒ£ Jangan Lupa Return

```javascript
// âœ… BENAR - Ada return
.map(word => word.toUpperCase())

// âŒ SALAH - Tidak ada return (hasilnya undefined)
.map(word => {
  word.toUpperCase();
})

// âœ… BENAR - Jika pakai kurung kurawal, harus return
.map(word => {
  return word.toUpperCase();
})
```

### 5ï¸âƒ£ Template String Lebih Rapi untuk String

```javascript
// âœ… BAIK
name: `${firstName} ${lastName}`

// âŒ KURANG RAPI
name: firstName + ' ' + lastName
```

### 6ï¸âƒ£ Reduce untuk Berbagai Use Case

```javascript
// Penjumlahan
.reduce((sum, num) => sum + num, 0)

// Perkalian
.reduce((product, num) => product * num, 1)

// Menghitung frekuensi
.reduce((count, item) => {
  count[item] = (count[item] || 0) + 1;
  return count;
}, {})

// Mengelompokkan data
.reduce((groups, item) => {
  const group = item.category;
  groups[group] = [...(groups[group] || []), item];
  return groups;
}, {})
```
