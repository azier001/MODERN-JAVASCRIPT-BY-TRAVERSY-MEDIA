<a name="apa-itu-foreach"></a>
## ğŸ“– Apa itu forEach()?

`forEach()` adalah sebuah **method** (fungsi bawaan) yang memungkinkan kamu untuk mengulang setiap element dalam array secara otomatis. Bayangkan seperti menunjuk ke setiap item dalam list dan melakukan sesuatu dengannya.

### Karakteristik Utama `forEach()` âœ¨

| Aspek | Keterangan |
|-------|-----------|
| **Fungsi** | Melakukan iterasi pada setiap element dalam array |
| **Return Value** | âŒ Tidak mengembalikan nilai apapun (`undefined`) |
| **Modifikasi Array** | Hanya melakukan aksi, tidak mengubah array |
| **Syntax** | Modern dan clean |

```javascript
// Loop biasa (cara lama)
const socials = ['Twitter', 'LinkedIn', 'Facebook'];
for (let i = 0; i < socials.length; i++) {
  console.log(socials[i]);
}

// Menggunakan forEach() (cara modern) âœ¨
socials.forEach((item) => console.log(item));
```

<a name="syntax-dan-penggunaan-dasar"></a>
## ğŸ’» Syntax dan Penggunaan Dasar

### Basic Structure

```javascript
array.forEach((parameter) => {
  // Kode yang akan dijalankan untuk setiap item
});
```

### Contoh Paling Sederhana

```javascript
const socials = ['Twitter', 'LinkedIn', 'Facebook', 'Instagram'];

// Menjalankan forEach()

  console.log(item);
});

// Output:
// Twitter
// LinkedIn
// Facebook
// Instagram
```

### Penjelasan Step by Step ğŸ”

1. **`socials.forEach(...)`** - Memanggil method forEach pada array `socials`
2. **`(item) => { ... }`** - Callback function dengan parameter `item`
3. **`item`** - Variable yang merepresentasikan setiap element dalam array (satu per satu)
4. **`console.log(item)`** - Kode yang dijalankan untuk setiap item

> **Penting!** Nama parameter tidak harus `item`. Kamu bisa menggunakan nama apapun yang meaningful!

## ğŸ Mengakses Index dan Array

Callback function di `forEach()` sebenarnya bisa menerima **3 parameter**:

```javascript
array.forEach((element, index, array) => {
  // element: item saat ini
  // index: posisi item (0, 1, 2, ...)
  // array: seluruh array
});
```

### Parameter Pertama: Element

```javascript
const socials = ['Twitter', 'LinkedIn', 'Facebook'];

socials.forEach((item) => {
  console.log(item); // 'Twitter', 'LinkedIn', 'Facebook'
});
```

### Parameter Kedua: Index ğŸ“

```javascript
const socials = ['Twitter', 'LinkedIn', 'Facebook'];

socials.forEach((item, index) => {
  console.log(`${index}: ${item}`);
});

// Output:
// 0: Twitter
// 1: LinkedIn
// 2: Facebook
```

**Catatan:** Index selalu dimulai dari **0** (nol), bukan 1!

### Parameter Ketiga: Entire Array ğŸ“¦

```javascript
const socials = ['Twitter', 'LinkedIn', 'Facebook'];

socials.forEach((item, index, arr) => {
  console.log(`${index} - ${item}`, arr);
});

// Output:
// 0 - Twitter [ 'Twitter', 'LinkedIn', 'Facebook' ]
// 1 - LinkedIn [ 'Twitter', 'LinkedIn', 'Facebook' ]
// 2 - Facebook [ 'Twitter', 'LinkedIn', 'Facebook' ]
```

### Kapan Menggunakan Ketiga Parameter? ğŸ¤”

| Kebutuhan | Parameter |
|-----------|-----------|
| Hanya akses item | `(item)` |
| Item + posisi | `(item, index)` |
| Item + posisi + array lain | `(item, index, arr)` |

## ğŸ·ï¸ Menggunakan Named Function

Daripada menulis function inline, kamu bisa membuat function terpisah dan menggunakannya sebagai callback:

### Step 1ï¸âƒ£: Buat Function Terpisah

### Step 2ï¸âƒ£: Gunakan Sebagai Callback

```javascript

// Twitter
// LinkedIn
```

### âŒ Kesalahan Umum

```javascript

// âœ… BENAR - Passing function reference
```

**Mengapa tidak boleh `()`?**
- `logSocial` = reference ke function (yang ingin kita pass)
- `logSocial()` = menjalankan function immediately dan pass hasilnya (bukan yang kita mau)

### Contoh Lengkap

