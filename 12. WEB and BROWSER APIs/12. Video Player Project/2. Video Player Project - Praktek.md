<a name="langkah-6-menampilkan-timestamp"></a>
### â±ï¸ Langkah 6: Menampilkan Timestamp

Sekarang kita tambahkan kode untuk menampilkan waktu dalam format `MM:SS`.

**Tambahkan di dalam fungsi `updateProgress()`**:

```javascript
function updateProgress() {
  // Update progress bar
  progress.value = (video.currentTime / video.duration) * 100;

  // ===== BAGIAN BARU: Timestamp =====
  
  // 1. Hitung MENIT
  let minutes = Math.floor(video.currentTime / 60);
  
  // 2. Tambahkan 0 di depan jika menit < 10
  if (minutes < 10) {
    minutes = '0' + String(minutes);
  }

  // 3. Hitung DETIK (sisa dari pembagian 60)
  let seconds = Math.floor(video.currentTime % 60);
  
  // 4. Tambahkan 0 di depan jika detik < 10
  if (seconds < 10) {
    seconds = '0' + String(seconds);
  }

  // 5. Tampilkan di element timestamp
  timestamp.innerHTML = `${minutes}:${seconds}`;
}
```

> ğŸ’¡ **Penjelasan Step by Step**:
> 
> 1. **`Math.floor(video.currentTime / 60)`**  
>    Contoh: 125 detik Ã· 60 = 2.08... â†’ dibulatkan ke bawah = **2 menit**
> 
> 2. **Tambah '0' di depan**  
>    Contoh: 2 â†’ '02' agar tampilan rapi
> 
> 3. **`video.currentTime % 60`** (modulus)  
>    Contoh: 125 detik % 60 = **5 detik** (sisa pembagian)
> 
> 4. **Tambah '0' di depan**  
>    Contoh: 5 â†’ '05'
> 
> 5. **Template literal `${}`**  
>    Menggabungkan string: `02:05`

### File: `script.js` (Versi Final)

```javascript
// ==========================================
// 1. AMBIL SEMUA ELEMENT
// ==========================================
const video = document.getElementById('video');
const play = document.getElementById('play');
const stop = document.getElementById('stop');
const progress = document.getElementById('progress');
const timestamp = document.getElementById('timestamp');

// ==========================================
// 2. FUNGSI PLAY/PAUSE
// ==========================================
function playPause() {
  if (video.paused) {
    video.play();
  } else {
    video.pause();
  }
}

// ==========================================
// 3. FUNGSI UPDATE ICON
// ==========================================
function updateIcon() {
  if (video.paused) {
    play.innerHTML = '<i class="fa fa-play fa-2x"></i>';
  } else {
    play.innerHTML = '<i class="fa fa-pause fa-2x"></i>';
  }
}

// ==========================================
// 4. FUNGSI STOP VIDEO
// ==========================================
function stopVideo() {
  video.currentTime = 0;
  video.pause();
}

// ==========================================
// 5. FUNGSI UPDATE PROGRESS & TIMESTAMP
// ==========================================
function updateProgress() {
  // Update progress bar
  progress.value = (video.currentTime / video.duration) * 100;

  // Hitung menit
  let minutes = Math.floor(video.currentTime / 60);
  if (minutes < 10) {
    minutes = '0' + String(minutes);
  }

  // Hitung detik
  let seconds = Math.floor(video.currentTime % 60);
  if (seconds < 10) {
    seconds = '0' + String(seconds);
  }

  // Tampilkan timestamp
  timestamp.innerHTML = `${minutes}:${seconds}`;
}

// ==========================================
// 6. FUNGSI SET PROGRESS MANUAL
// ==========================================
function setProgress() {
  video.currentTime = (+progress.value * video.duration) / 100;
}

// ==========================================
// 7. EVENT LISTENERS
// ==========================================

// Event untuk video
video.addEventListener('click', playPause);
video.addEventListener('play', updateIcon);
video.addEventListener('pause', updateIcon);
video.addEventListener('timeupdate', updateProgress);

// Event untuk buttons
play.addEventListener('click', playPause);
stop.addEventListener('click', stopVideo);

// Event untuk progress bar
progress.addEventListener('click', setProgress);
```

## ğŸ“ Catatan Penting

### ğŸ¯ Perbedaan dengan Audio Player

| Aspek | Audio Player | Video Player (Kita) |
|-------|-------------|---------------------|
| **Progress Bar** | Menggunakan `<div>` dengan width CSS | Menggunakan `<input type="range">` |
| **Cara Update** | Set `width` style | Set `value` property |
| **Kelebihan** | Lebih smooth | Lebih mudah (built-in) |

### âš ï¸ Known Issue

Progress bar bisa sedikit "wonky" (tidak mulus) karena kita menggunakan `<input type="range">`. Untuk production, lebih baik gunakan `<div>` seperti di Audio Player.

### ğŸ”‘ Property Video Penting

| Property | Fungsi |
|----------|--------|
| `video.paused` | Cek apakah video pause (true/false) |
| `video.currentTime` | Get/Set waktu saat ini (detik) |
| `video.duration` | Total durasi video (detik) |
| `video.play()` | Method untuk play video |
| `video.pause()` | Method untuk pause video |

### ğŸª Event Video Penting

| Event | Kapan Di-trigger |
|-------|------------------|
| `'click'` | Saat element diklik |
| `'play'` | Saat video mulai play |
| `'pause'` | Saat video di-pause |
| `'timeupdate'` | Saat waktu video berubah (berkali-kali) |
