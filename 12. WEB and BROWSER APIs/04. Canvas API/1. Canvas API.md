<a name="apa-itu-canvas-api"></a>
## ğŸ“– Apa itu Canvas API?

Canvas API adalah **antarmuka pemrograman** yang berbasis pada **2D rendering context**. Context ini dibuat oleh elemen `<canvas>` di HTML, yang memungkinkan kita menggambar berbagai bentuk, teks, dan gambar.

### ğŸ”‘ Konsep Penting:
- **Canvas Element**: Tag HTML `<canvas>` sebagai area gambar
- **Context**: Objek yang digunakan untuk menggambar (ctx)
- **Koordinat**: Sistem X (horizontal) dan Y (vertikal) dari pojok kiri atas

## â­• Menggambar Circle (Lingkaran)

### Kode:

### ğŸ’¡ Penjelasan:

- **startAngle = 0**: Sudut awal (0 radian)

**`fill()`**: Mengisi bentuk dengan warna sesuai `fillStyle`

### ğŸ”¢ Tentang Radian:
- Lingkaran penuh = **2Ï€ radian** = **360 derajat**
- Setengah lingkaran = **Ï€ radian** = **180 derajat**
- `Math.PI` memberikan nilai Ï€ (3.14159...)

> ğŸ’¡ **Tips**: Jika tidak mengatur `fillStyle` baru, canvas akan menggunakan `fillStyle` terakhir (dalam kasus ini hijau dari rectangle).

<a name="menggambar-lines"></a>
## ğŸ“ Menggambar Lines (Garis)

### Kode:

```javascript
// Draw lines
ctx.beginPath();
ctx.strokeStyle = 'orange';
ctx.lineWidth = 5;
ctx.moveTo(10, 10);
ctx.lineTo(300, 300);
ctx.stroke();
```

### ğŸ’¡ Penjelasan Detail:

**`beginPath()`**: Memulai path baru
- Seperti mengangkat pena dari kertas dan mulai menggambar dari awal

**`strokeStyle`**: Mengatur warna garis/outline
- Berbeda dengan `fillStyle` yang untuk isian
- `strokeStyle` untuk garis, border, atau outline

**`lineWidth`**: Mengatur ketebalan garis
- Angka dalam pixel (contoh: 5 = 5 pixel tebal)

**`moveTo(x, y)`**: Memindahkan "pena" ke posisi tanpa menggambar
- `moveTo(10, 10)`: Pindah ke koordinat (10, 10)
- Tidak menggambar apa-apa, hanya mengatur titik awal

**`lineTo(x, y)`**: Menggambar garis dari posisi sekarang ke posisi baru
- `lineTo(300, 300)`: Menggambar garis ke koordinat (300, 300)

**`stroke()`**: Menggambar garis di canvas
- Tanpa `stroke()`, garis tidak akan terlihat
- `stroke()` adalah perintah untuk "render" garis tersebut

### ğŸ¯ Analogi:
```
beginPath()   â†’ Siapkan kertas baru
moveTo()      â†’ Letakkan pena di titik awal (tanpa menggambar)
lineTo()      â†’ Tarik pena ke titik tujuan (menggambar garis)
stroke()      â†’ Jadikan garis permanen/terlihat
```

<a name="menggambar-text"></a>
## âœï¸ Menggambar Text (Teks)

Canvas dapat menggambar teks dengan dua cara: **filled text** (terisi) dan **stroked text** (outline).

### Kode:

```javascript
// Draw text
ctx.font = '30px Arial';
ctx.lineWidth = 1;
ctx.fillStyle = 'blue';
ctx.strokeStyle = 'blue';
ctx.fillText('Hello World', 300, 100, 300);
ctx.strokeText('Hello World', 300, 500, 300);
```

### ğŸ’¡ Penjelasan:

**`font`**: Mengatur ukuran dan jenis font
- Format: `'[ukuran] [jenis-font]'`
- Contoh: `'30px Arial'`, `'20px Georgia'`

**`fillText(text, x, y, maxWidth)`**: Menggambar teks terisi
- **text**: String yang ingin ditampilkan
- **x = 300**: Posisi horizontal teks
- **y = 100**: Posisi vertikal teks
- **maxWidth = 300**: Lebar maksimal teks (akan menyusut jika terlalu panjang)

**`strokeText(text, x, y, maxWidth)`**: Menggambar teks outline
- Parameter sama dengan `fillText`
- Hanya menggambar garis tepi huruf (seperti font berongga)

### ğŸ¨ Perbedaan `fillText` vs `strokeText`:

| Properti | fillText | strokeText |
|----------|----------|------------|
| **Tampilan** | Huruf penuh terisi warna | Hanya outline/garis tepi huruf |
| **Warna dari** | `fillStyle` | `strokeStyle` |
| **Ketebalan** | Penuh | Mengikuti `lineWidth` |

> ğŸ’¡ **Catatan**: `lineWidth` sebelumnya masih 5, makanya `strokeText` terlihat tebal. Kita ubah jadi 1 agar lebih rapi.

<a name="menggambar-image"></a>
## ğŸ–¼ï¸ Menggambar Image (Gambar)

Canvas dapat menampilkan gambar eksternal seperti PNG, JPG, atau format lainnya.

### Kode:

```javascript

image.style.display = 'none';

  ctx.drawImage(image, 250, 250, 100, 100)
});
```

### ğŸ’¡ Penjelasan Detail:

**`querySelector('img')`**: Mengambil elemen `<img>` dari HTML
- Memilih gambar pertama yang ditemukan di halaman

**`image.style.display = 'none'`**: Menyembunyikan gambar asli
- Gambar tidak ditampilkan di halaman HTML biasa
- Hanya akan ditampilkan di dalam canvas

**`addEventListener('load', ...)`**: Menunggu gambar selesai dimuat
- âš ï¸ **Penting**: Karena gambar adalah resource eksternal, kita harus menunggu sampai benar-benar dimuat
- Jika langsung menggambar tanpa menunggu, gambar mungkin belum siap dan tidak muncul

**`drawImage(image, x, y, width, height)`**: Menggambar gambar ke canvas
- **image**: Elemen gambar yang ingin digambar
- **x = 250**: Posisi horizontal gambar
- **y = 250**: Posisi vertikal gambar
- **width = 100**: Lebar gambar di canvas
- **height = 100**: Tinggi gambar di canvas

### âš¡ Kenapa Perlu `addEventListener('load')`?

```javascript
// âŒ SALAH - Gambar mungkin belum dimuat

// âœ… BENAR - Tunggu sampai gambar dimuat
image.addEventListener('load', () => {
  ctx.drawImage(image, 250, 250, 100, 100);
});
```
