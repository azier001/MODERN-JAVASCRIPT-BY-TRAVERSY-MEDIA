## ğŸ”´ Langkah 2: Menggambar Lingkaran

Untuk lingkaran, kita menggunakan method `arc()`.

### Kode:

```javascript

```

### ğŸ“Œ Penjelasan:

| Kode | Fungsi |
|------|--------|
| `ctx.fillStyle = 'red'` | Ubah warna menjadi merah |
| `ctx.arc()` | Menggambar arc/lingkaran |
| `ctx.fill()` | Mengisi bentuk dengan warna |

**Parameter `arc(x, y, radius, startAngle, endAngle)`:**
- `x = 300` â†’ Posisi tengah lingkaran dari kiri
- `y = 300` â†’ Posisi tengah lingkaran dari atas
- `radius = 100` â†’ Jari-jari lingkaran
- `startAngle = 0` â†’ Sudut awal (0 radian)
- `endAngle = Math.PI * 2` â†’ Sudut akhir (360Â°)

### ğŸ§® Kenapa `Math.PI * 2`?
- Satu lingkaran penuh = 2Ï€ radian (sekitar 6.28)
- `Math.PI` = 3.14...
- `Math.PI * 2` = lingkaran penuh!

> **âš ï¸ Penting**: Jangan lupa panggil `ctx.fill()` atau lingkaran tidak akan terlihat!

<a name="langkah-3-menggambar-garis"></a>
## ğŸ“ Langkah 3: Menggambar Garis

Garis menggunakan konsep "path" (jalur).

### Kode:

```javascript
// Draw lines
ctx.beginPath();
ctx.strokeStyle = 'orange';
ctx.lineWidth = 5;
ctx.moveTo(10, 10);
ctx.lineTo(300, 300);
ctx.stroke();
```

### ğŸ“Œ Penjelasan Step-by-Step:

1. **`ctx.beginPath()`** â†’ Mulai jalur baru
2. **`ctx.strokeStyle = 'orange'`** â†’ Warna garis oranye
3. **`ctx.lineWidth = 5`** â†’ Ketebalan garis 5 pixel
4. **`ctx.moveTo(10, 10)`** â†’ Pindahkan "pena" ke titik (10, 10)
5. **`ctx.lineTo(300, 300)`** â†’ Gambar garis ke titik (300, 300)
6. **`ctx.stroke()`** â†’ Tampilkan garisnya!

### ğŸ¯ Perbedaan Penting:

| Method | Untuk |
|--------|-------|
| `fill()` | Mengisi bentuk dengan warna |
| `stroke()` | Menggambar garis/outline |
| `fillStyle` | Warna isian |
| `strokeStyle` | Warna garis |

> **ğŸ’¡ Analogi**: Bayangkan `moveTo()` seperti mengangkat pena, lalu `lineTo()` seperti menggambar garis sambil menekan pena!

<a name="langkah-4-menulis-teks"></a>
## âœï¸ Langkah 4: Menulis Teks

Canvas bisa menulis teks dengan 2 cara: **terisi** atau **outline**.

### Kode:

```javascript
// Draw text
ctx.font = '30px Arial';
ctx.lineWidth = 1;
ctx.fillStyle = 'blue';
ctx.strokeStyle = 'blue';

// Teks terisi
ctx.fillText('Hello World', 300, 100, 300);

// Teks outline
ctx.strokeText('Hello World', 300, 500, 300);
```

### ğŸ“Œ Penjelasan:

**Setup Font:**
- `ctx.font = '30px Arial'` â†’ Ukuran 30px, font Arial

**Teks Terisi (fillText):**
- Parameter: `fillText(text, x, y, maxWidth)`
- `'Hello World'` â†’ Teks yang ditampilkan
- `300, 100` â†’ Posisi x dan y
- `300` â†’ Lebar maksimal teks

**Teks Outline (strokeText):**
- Sama seperti `fillText`, tapi hanya outline
- `lineWidth` mengatur ketebalan outline

### ğŸ¨ Perbedaan Visual:

| Method | Hasil |
|--------|-------|
| `fillText()` | Teks terisi penuh warna biru |
| `strokeText()` | Teks dengan border biru saja |

## ğŸ–¼ï¸ Langkah 5: Menampilkan Gambar

Kita bisa memasukkan gambar eksternal ke dalam canvas!

### Kode:


### ğŸ“Œ Penjelasan Step-by-Step:

1. **`document.querySelector('img')`** â†’ Ambil elemen gambar dari HTML
2. **`image.style.display = 'none'`** â†’ Sembunyikan gambar dari halaman
3. **`image.addEventListener('load', ...)`** â†’ Tunggu gambar selesai loading
4. **`ctx.drawImage(image, 250, 250, 100, 100)`** â†’ Gambar di canvas

**Parameter `drawImage(image, x, y, width, height)`:**
- `image` â†’ Elemen gambar
- `250, 250` â†’ Posisi x, y
- `100, 100` â†’ Ukuran gambar di canvas

### âš ï¸ Mengapa Pakai Event Listener?

```javascript
// âŒ SALAH - Gambar belum loaded

// âœ… BENAR - Tunggu gambar loaded dulu
image.addEventListener('load', () => {
    ctx.drawImage(image, 250, 250, 100, 100);
});
```

> **ğŸ’¡ Tips**: Gambar eksternal butuh waktu loading, jadi kita tunggu event `load` agar tidak error!
