<a name="apa-itu-html-canvas"></a>
## ğŸ“– Apa itu HTML Canvas?

### Pengertian Dasar

**Canvas** adalah elemen HTML yang memungkinkan kita untuk **menggambar grafis menggunakan JavaScript**. Bayangkan canvas seperti kanvas putih di mana kita bisa melukis apa saja dengan kode!

```html
<!-- Ini adalah canvas -->
<canvas id="canvas" width="500" height="500"></canvas>
```

### Properti Canvas

| Properti | Penjelasan |
|----------|-----------|
| `id` | Identitas unik untuk mengakses canvas dari JavaScript |
| `width` | Lebar canvas (dalam pixel) |
| `height` | Tinggi canvas (dalam pixel) |

### Cara Menggunakan Canvas

```javascript
// 1. Ambil element canvas
const canvas = document.getElementById('canvas');

// 2. Dapatkan konteks 2D (untuk menggambar)
const ctx = canvas.getContext('2d');

// 3. Mulai menggambar!
ctx.fillStyle = 'blue';  // Set warna
ctx.fillRect(0, 0, 100, 100);  // Gambar persegi
```

### Metode Canvas yang Penting

```javascript
// Membersihkan canvas
ctx.clearRect(0, 0, width, height);

// Menggeser titik awal (0,0) ke lokasi baru
ctx.translate(x, y);

// Memutar canvas
ctx.rotate(angleInRadians);

// Menyimpan state canvas saat ini
ctx.save();

// Mengembalikan state canvas ke kondisi sebelumnya
ctx.restore();
```

### Cara Mengambil Nilai Warna

## ğŸ¨ Langkah 3: Mengubah Warna Border Jam

Mari kita mulai mengubah warna! Langkah pertama adalah **border (tepi) jam**.

### ğŸ§  Konsep

Dalam canvas, kita gunakan properti `strokeStyle` untuk mengatur warna garis/border:

```javascript

```

### ğŸ“ Kode Lengkap `script.js`

Ganti semua kode di `script.js` dengan kode berikut:

```javascript

// LANGKAH 2: Ambil semua element
// ========================================

const canvas = document.getElementById('canvas');
const faceColor = document.getElementById('face-color');
const borderColor = document.getElementById('border-color');
const lineColor = document.getElementById('line-color');
const largeHandColor = document.getElementById('large-hand-color');
const secondHandColor = document.getElementById('second-hand-color');

// ========================================
// FUNGSI UTAMA: Menggambar Jam
// ========================================

function clock() {
  const now = new Date();
  const ctx = canvas.getContext('2d');

  // Setup canvas
  ctx.save(); // Simpan state default
  ctx.clearRect(0, 0, 500, 500);  // Bersihkan canvas
  ctx.translate(250, 250);  // Geser titik (0,0) ke tengah
  ctx.rotate(-Math.PI / 2);  // Putar canvas -90 derajat

  // Set default styles
  ctx.strokeStyle = '#000000';
  ctx.fillStyle = '#f4f4f4';
  ctx.lineWidth = 5;
  ctx.lineCap = 'round';

  // ========================================
  // LANGKAH 3: Gambar Border/Tepi Jam
  // ========================================
  ctx.save();
  ctx.beginPath();
  ctx.lineWidth = 14;
  ctx.strokeStyle = borderColor.value;  // â­ GUNAKAN WARNA DARI INPUT!
  ctx.fillStyle = faceColor.value;
  ctx.arc(0, 0, 142, 0, Math.PI * 2, true);
  ctx.stroke();
  ctx.fill();
  ctx.restore();

  ctx.restore(); // Kembalikan state default

  requestAnimationFrame(clock);
}

requestAnimationFrame(clock);
```

<a name="langkah-5-mengubah-warna-garis-angka"></a>
## ğŸ¨ Langkah 5: Mengubah Warna Garis Angka

Sekarang mari kita ubah warna **garis penunjuk angka jam**!

### ğŸ§  Konsep

Garis angka dibuat dengan 2 loop:
- Loop pertama: Garis jam (12, 1, 2, 3, dst)
- Loop kedua: Garis menit (lebih pendek)

Kita ubah `strokeStyle` untuk kedua loop ini.

### ğŸ“ Kode Lengkap `script.js`

```javascript
const canvas = document.getElementById('canvas');
const faceColor = document.getElementById('face-color');
const borderColor = document.getElementById('border-color');
const lineColor = document.getElementById('line-color');
const largeHandColor = document.getElementById('large-hand-color');
const secondHandColor = document.getElementById('second-hand-color');

function clock() {
  const now = new Date();
  const ctx = canvas.getContext('2d');

  ctx.save();
  ctx.clearRect(0, 0, 500, 500);
  ctx.translate(250, 250);
  ctx.rotate(-Math.PI / 2);

  ctx.strokeStyle = '#000000';
  ctx.fillStyle = '#f4f4f4';
  ctx.lineWidth = 5;
  ctx.lineCap = 'round';

  // Gambar border & face
  ctx.save();
  ctx.beginPath();
  ctx.lineWidth = 14;
  ctx.strokeStyle = borderColor.value;
  ctx.fillStyle = faceColor.value;
  ctx.arc(0, 0, 142, 0, Math.PI * 2, true);
  ctx.stroke();
  ctx.fill();
  ctx.restore();

  // ========================================
  // LANGKAH 5: Gambar Garis Jam (Hour Lines)
  // ========================================
  ctx.save();
  ctx.strokeStyle = lineColor.value;  // â­ GUNAKAN WARNA DARI INPUT!
  for (let i = 0; i < 12; i++) {
    ctx.beginPath();
    ctx.rotate(Math.PI / 6);  // Putar 30 derajat setiap iterasi
    ctx.moveTo(100, 0);  // Mulai dari koordinat (100, 0)
    ctx.lineTo(120, 0);  // Garis ke (120, 0)
    ctx.stroke();  // Gambar garis
  }
  ctx.restore();

  // ========================================
  // LANGKAH 5: Gambar Garis Menit (Minute Lines)
  // ========================================
  ctx.save();
  ctx.strokeStyle = lineColor.value;  // â­ GUNAKAN WARNA DARI INPUT!
  ctx.lineWidth = 4;
  for (let i = 0; i < 60; i++) {
    if (i % 5 !== 0) {  // Abaikan kelipatan 5 (sudah ada garis jam)
      ctx.beginPath();
      ctx.moveTo(117, 0);
      ctx.lineTo(120, 0);
      ctx.stroke();
    }
    ctx.rotate(Math.PI / 30);  // Putar 6 derajat setiap iterasi
  }
  ctx.restore();

  ctx.restore();

  requestAnimationFrame(clock);
}
```

## ğŸ¨ Langkah 6: Mengubah Warna Jarum Besar

Mari kita ubah warna **jarum jam dan jarum menit** (jarum besar)!

### ğŸ§  Konsep

Jarum besar terdiri dari 2 jarum:
1. **Hour Hand** - Jarum jam (lebih pendek)
2. **Minute Hand** - Jarum menit (lebih panjang)

Kedua jarum menggunakan properti `strokeStyle`.

### ğŸ“ Kode Lengkap `script.js`

```javascript

const faceColor = document.getElementById('face-color');
const borderColor = document.getElementById('border-color');
const lineColor = document.getElementById('line-color');
const largeHandColor = document.getElementById('large-hand-color');
const secondHandColor = document.getElementById('second-hand-color');

function clock() {
  const now = new Date();
  const ctx = canvas.getContext('2d');

  ctx.save();
  ctx.clearRect(0, 0, 500, 500);
  ctx.translate(250, 250);
  ctx.rotate(-Math.PI / 2);

  ctx.strokeStyle = '#000000';
  ctx.fillStyle = '#f4f4f4';
  ctx.lineWidth = 5;
  ctx.lineCap = 'round';

  // Gambar border & face
  ctx.save();
  ctx.beginPath();
  ctx.lineWidth = 14;
  ctx.strokeStyle = borderColor.value;
  ctx.fillStyle = faceColor.value;
  ctx.arc(0, 0, 142, 0, Math.PI * 2, true);
  ctx.stroke();
  ctx.fill();
  ctx.restore();

  // Gambar garis jam
  ctx.save();
  ctx.strokeStyle = lineColor.value;
  for (let i = 0; i < 12; i++) {
    ctx.beginPath();
    ctx.rotate(Math.PI / 6);
    ctx.moveTo(100, 0);
    ctx.lineTo(120, 0);
    ctx.stroke();
  }

  ctx.save();
  ctx.strokeStyle = lineColor.value;

    if (i % 5 !== 0) {
      ctx.beginPath();
      ctx.moveTo(117, 0);
      ctx.lineTo(120, 0);
      ctx.stroke();
    }

  // ========================================
```

## ğŸ¨ Langkah 7: Mengubah Warna Jarum Detik

Langkah terakhir sebelum save! Mari kita ubah warna **jarum detik** dan **lingkaran tengah**!

### ğŸ§  Konsep

Jarum detik terdiri dari:
1. **Stroke** - Garis jarum detik
2. **Fill** - Lingkaran kecil di tengah jam

Kedua elemen menggunakan warna yang sama dari `secondHandColor`.

### ğŸ“ Kode Lengkap `script.js`

```javascript

const faceColor = document.getElementById('face-color');
const borderColor = document.getElementById('border-color');
const lineColor = document.getElementById('line-color');
const largeHandColor = document.getElementById('large-hand-color');
const secondHandColor = document.getElementById('second-hand-color');

function clock() {
  const now = new Date();
  const ctx = canvas.getContext('2d');

  ctx.save();
  ctx.clearRect(0, 0, 500, 500);
  ctx.translate(250, 250);
  ctx.rotate(-Math.PI / 2);

  ctx.strokeStyle = '#000000';
  ctx.fillStyle = '#f4f4f4';
  ctx.lineWidth = 5;
  ctx.lineCap = 'round';

  // Gambar border & face
  ctx.save();
  ctx.beginPath();
  ctx.lineWidth = 14;
  ctx.strokeStyle = borderColor.value;
  ctx.fillStyle = faceColor.value;
  ctx.arc(0, 0, 142, 0, Math.PI * 2, true);
  ctx.stroke();
  ctx.fill();
  ctx.restore();

  // Gambar garis jam
  ctx.save();
  ctx.strokeStyle = lineColor.value;
  for (let i = 0; i < 12; i++) {
    ctx.beginPath();
    ctx.rotate(Math.PI / 6);
    ctx.moveTo(100, 0);
    ctx.lineTo(120, 0);
    ctx.stroke();
  }
  ctx.restore();

  // Gambar garis menit
  ctx.save();
  ctx.strokeStyle = lineColor.value;
  ctx.lineWidth = 4;
  for (let i = 0; i < 60; i++) {
    if (i % 5 !== 0) {
      ctx.beginPath();
      ctx.moveTo(117, 0);
      ctx.lineTo(120, 0);
      ctx.stroke();
    }
    ctx.rotate(Math.PI / 30);
  }
  ctx.restore();

  // Ambil jam sekarang
  const hr = now.getHours() % 12;
  const min = now.getMinutes();
  const sec = now.getSeconds();

  // Gambar jarum jam
  ctx.save();
  ctx.rotate(
    (Math.PI / 6) * hr + (Math.PI / 360) * min + (Math.PI / 21600) * sec
  );
  ctx.strokeStyle = largeHandColor.value;
  ctx.lineWidth = 14;
  ctx.beginPath();
  ctx.moveTo(-20, 0);
  ctx.lineTo(80, 0);
  ctx.stroke();
  ctx.restore();

  // Gambar jarum menit
  ctx.save();
  ctx.rotate((Math.PI / 30) * min + (Math.PI / 1800) * sec);
  ctx.strokeStyle = largeHandColor.value;
  ctx.lineWidth = 10;
  ctx.beginPath();
  ctx.moveTo(-28, 0);
  ctx.lineTo(112, 0);
  ctx.stroke();
  ctx.restore();

  // ========================================
  // LANGKAH 7: Gambar Jarum Detik (Second Hand)
  // ========================================
  ctx.save();
  ctx.rotate((sec * Math.PI) / 30);  // Putar 6Â° per detik
  ctx.strokeStyle = secondHandColor.value;  // â­ GUNAKAN WARNA DARI INPUT!
  ctx.fillStyle = secondHandColor.value;     // â­ GUNAKAN WARNA DARI INPUT!
  ctx.lineWidth = 6;
  
  // Gambar garis jarum detik
  ctx.beginPath();
  ctx.moveTo(-30, 0);
  ctx.lineTo(100, 0);
  ctx.stroke();
  
  // Gambar lingkaran tengah
  ctx.beginPath();
  ctx.arc(0, 0, 10, 0, Math.PI * 2, true);
  ctx.fill();
  
  ctx.restore();

  ctx.restore();

  requestAnimationFrame(clock);
}

requestAnimationFrame(clock);
```

### ğŸ“ Kode Lengkap `script.js`

```javascript
  // Konversi canvas menjadi data URL (image/png)

  // Buat element link secara dinamis
  const link = document.createElement('a');
  
  // Set nama file yang akan di-download
  link.download = 'clock.png';
  
  // Set href link ke data URL
  link.href = dataURL;
  
  // "Klik" link secara otomatis untuk trigger download
  link.click();
  
  console.log('âœ… Jam berhasil disimpan sebagai clock.png!');
});
```
