## ğŸ¥ Apa itu Drum Machine?

Drum Machine adalah alat musik elektronik yang menghasilkan suara drum. Dalam project ini, kita membuat versi web sederhana di mana:

- ğŸ¹ **Keyboard A-K** = Tombol drum yang berbeda

**Suara yang tersedia:**

- **S** = Hi-hat
- **D** = Kick (Bass drum)
- **F** = Open hat
- **G** = Boom
- **H** = Ride

- **K** = Tom

<a name="34-validasi-audio"></a>
### 3.4 Validasi Audio

Bagaimana jika user menekan tombol yang tidak ada audionya? Misalnya tombol **R** atau **T**?

```javascript

```

**ğŸ’¡ Penjelasan:**
- `if (!audio)` â†’ Jika audio **tidak** ada (null)
- `return;` â†’ Hentikan function, jangan jalankan kode selanjutnya

**ğŸ“Œ Mengapa pakai `!` (tanda seru)?**
- `!` artinya **NOT** (kebalikan)
- `!audio` â†’ "jika audio tidak ada"

**ğŸ§ª Testing:**
1. Tekan tombol **A** â†’ Console muncul "Audio ditemukan!"
2. Tekan tombol **R** â†’ Tidak ada yang muncul (karena di-return)

<a name="36-menambahkan-animasi"></a>
### 3.6 Menambahkan Animasi

Terakhir, kita tambahkan efek visual saat tombol ditekan! **Update function menjadi:**

```javascript

  const audio = document.querySelector(`audio[data-key="${e.keyCode}"]`);
  const key = document.querySelector(`.key[data-key="${e.keyCode}"]`);
  
  if (!audio) return;
  
  audio.currentTime = 0;

  }, 100);
}
```

**ğŸ’¡ Penjelasan:**

**Baris 1:**
```javascript

```
- `classList` â†’ Daftar class pada element
- `.add('playing')` â†’ Menambahkan class "playing"
- Ingat CSS kita? Class `.playing` membuat tombol membesar!

**Baris 2-4:**
```javascript

  key.classList.remove('playing');
}, 100);
```
- `setTimeout()` â†’ Function yang menjalankan kode setelah delay tertentu
- `() => { }` â†’ Arrow function (function tanpa nama)
- `.remove('playing')` â†’ Menghapus class "playing"
- `100` â†’ Delay 100 millisecond (0.1 detik)

**ğŸ¯ Cara Kerjanya:**
1. Tombol ditekan â†’ Class "playing" ditambahkan â†’ Tombol membesar
2. Tunggu 100ms
3. Class "playing" dihapus â†’ Tombol kembali normal

**Hasilnya:** Efek "flash" yang cepat! âœ¨
