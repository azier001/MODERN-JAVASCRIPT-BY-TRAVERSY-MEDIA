### ğŸ“– Cara Destructuring Array

Array juga bisa di-destructuring, tapi caranya **berbeda** dengan object:

**ğŸ’¡ Penjelasan:**
- Array diambil **berdasarkan urutan** (index)

### ğŸ’ Rest Operator (`...`)

Bagaimana jika kita ingin mengambil **sisa elemen** yang belum diambil? Gunakan **rest operator** (`...`)!


**ğŸ’¡ Penjelasan:**
- `...rest` = Ambil **semua elemen sisanya** dan masukkan ke array baru
- Nama variable bebas, tidak harus `rest` (bisa `...nums`, `...others`, dll)

#### Contoh dengan Nama Berbeda:

**ğŸ¤” Rest Operator vs Spread Operator:**
- Keduanya menggunakan simbol yang sama: `...`
- **Rest Operator** = Mengumpulkan sisa elemen menjadi satu array (digunakan saat destructuring)
- **Spread Operator** = Menyebarkan elemen array (digunakan saat menyalin atau menggabungkan array)
