## ğŸ¯ Pengenalan Object Literals

Object Literals adalah salah satu **data structure** penting dalam JavaScript yang memungkinkan kita menyimpan data dalam bentuk **key-value pairs** (pasangan kunci-nilai).

**Analogi Sederhana**: 
Bayangkan object seperti kartu identitas seseorang yang memiliki berbagai informasi seperti nama, umur, alamat, dll. Setiap informasi memiliki label (key) dan isinya (value).

---

<a name="apa-itu-object-literals"></a>
## ğŸ“– Apa itu Object Literals?

Object Literals adalah cara untuk membuat object menggunakan **curly braces** `{}` dengan format:
- **Key** (kunci/label) di sebelah kiri
- **Colon** `:` sebagai pemisah
- **Value** (nilai) di sebelah kanan
- **Comma** `,` untuk memisahkan setiap pasangan key-value

### ğŸ” Struktur Dasar

```javascript

  key1: value1,
  key2: value2,
  key3: value3
}
```

### âœ¨ Key dan Properties itu Sama

- **Key** = **Property** (dua istilah yang sama)
- Contoh: `name` adalah key sekaligus property
