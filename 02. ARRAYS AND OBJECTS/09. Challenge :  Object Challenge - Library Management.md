## ğŸ Step 3: Destructuring dan Rename Variable

<a name="penjelasan-konsep-step-3"></a>
### ğŸ“ Penjelasan Konsep

**Destructuring** adalah cara modern dan ringkas untuk "membongkar" nilai dari objek atau array. Bayangkan seperti membuka bungkus kado dan langsung mengambil isinya!

**Cara Lama:**
```javascript

```

**Cara Baru (Destructuring):**
```javascript

```

<a name="kode-praktek-step-3"></a>
### ğŸ’» Kode Praktek

```javascript

console.log(firstBook);  // Output: The Road Ahead
```

<a name="penjelasan-detail-step-3"></a>
### ğŸ” Penjelasan Detail

Mari kita bedah `const { title: firstBook } = library[0];`:

| Bagian | Penjelasan |
|--------|------------|
| `const { ... }` | Syntax destructuring untuk objek (pakai kurung kurawal) |
| `title` | Properti yang ingin kita ambil dari objek |
| `: firstBook` | **Rename** - mengubah nama variable dari `title` menjadi `firstBook` |
| `= library[0]` | Objek sumber yang akan di-destructure (buku pertama) |

**Perbandingan:**

| Tanpa Destructuring | Dengan Destructuring |
|---------------------|----------------------|
| `const title = library[0].title;` | `const { title } = library[0];` |
| `const titleRenamed = library[0].title;` | `const { title: titleRenamed } = library[0];` |

**Keuntungan Destructuring:**
- âœ… Kode lebih ringkas
- âœ… Lebih mudah dibaca
- âœ… Bisa rename variable sekaligus
- âœ… Bisa ambil banyak properti sekaligus

**Contoh Ambil Banyak Properti:**
```javascript
const { title, author } = library[0];
console.log(title);   // The Road Ahead
console.log(author);  // Bill Gates
```
