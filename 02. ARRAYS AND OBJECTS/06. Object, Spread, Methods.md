> âš ï¸ **Perhatian**: Kedua cara ini menghasilkan hasil yang sama. Object literal (`{}`) lebih modern dan lebih direkomendasikan.

<a name="object-nesting"></a>
## ğŸ Object Nesting (Bersarang)

Object dapat berisi object lain di dalamnya. Ini disebut **nesting** atau **nested object**.

```javascript
const person = {
  address: {
    coords: {
      lat: 42.9384,
      lng: -71.3232,
    },
  },
};

// Mengakses nilai latitude
x = person.address.coords.lat;
console.log(x); // Output: 42.9384
```

**Penjelasan:**
- Object `person` memiliki properti `address`
- `address` berisi object lain dengan properti `coords`
- `coords` berisi object dengan properti `lat` dan `lng`
- Untuk mengakses nilai, gunakan dot notation secara bertingkat: `person.address.coords.lat`

> ğŸ’¡ **Tips**: Nested object berguna untuk merepresentasikan struktur data yang kompleks dan hierarkis.

<a name="spread-operator"></a>
## âœ¨ Spread Operator pada Object

Spread operator (`...`) digunakan untuk menggabungkan properti dari beberapa object.

```javascript
const obj1 = { a: 1, b: 2 };
const obj2 = { c: 3, d: 4 };

// Menggabungkan obj1 dan obj2
const obj3 = { ...obj1, ...obj2 };
console.log(obj3);
// Output: { a: 1, b: 2, c: 3, d: 4 }
```

**Penjelasan:**
- `...obj1` menyebarkan semua properti dari `obj1`
- `...obj2` menyebarkan semua properti dari `obj2`
- Hasilnya adalah object baru yang berisi semua properti dari kedua object
- Properti dari `obj1` dan `obj2` berada di level yang sama (tidak nested)

### ğŸ”„ Perbedaan Dengan dan Tanpa Spread

```javascript
// âŒ TANPA spread operator
const obj3 = { obj1, obj2 };
// Output: { obj1: {a: 1, b: 2}, obj2: {c: 3, d: 4} }
// Object menjadi nested

// âœ… DENGAN spread operator
const obj3 = { ...obj1, ...obj2 };
// Output: { a: 1, b: 2, c: 3, d: 4 }
// Properti digabungkan di level yang sama
```

<a name="object-assign"></a>
## ğŸ”€ Method Object.assign()

Method `Object.assign()` adalah cara lama untuk menggabungkan object sebelum spread operator diperkenalkan.

```javascript
const obj1 = { a: 1, b: 2 };
const obj2 = { c: 3, d: 4 };

// Menggunakan Object.assign()
const obj4 = Object.assign({}, obj1, obj2);
console.log(obj4);
// Output: { a: 1, b: 2, c: 3, d: 4 }
```

**Penjelasan:**
- Parameter pertama `{}` adalah target object (object tujuan)
- Parameter kedua dan seterusnya adalah source object (object sumber)
- Semua properti dari source object akan disalin ke target object
- Hasilnya sama dengan menggunakan spread operator

> ğŸ’¡ **Catatan**: Spread operator (`...`) lebih modern dan lebih mudah dibaca, tapi `Object.assign()` masih berguna dalam beberapa kasus.

<a name="array-of-objects"></a>
## ğŸ“¦ Array of Objects

Dalam aplikasi nyata, kita sering bekerja dengan array yang berisi banyak object (misalnya: daftar pengguna, to-do list, produk, dll).

```javascript
const todos = [
  { id: 1, name: 'Buy Milk' },
  { id: 2, name: 'Pickup kids from school' },
  { id: 3, name: 'Take out trash' },
];

// Mengakses object pertama
x = todos[0];
console.log(x); // Output: { id: 1, name: 'Buy Milk' }

// Mengakses properti dari object pertama
x = todos[0].name;
console.log(x); // Output: 'Buy Milk'
```

**Penjelasan:**
- `todos` adalah array yang berisi 3 object
- Setiap object merepresentasikan satu to-do item
- `todos[0]` mengakses object pertama (index 0)
- `todos[0].name` mengakses properti `name` dari object pertama

> ğŸ¯ **Use Case**: Array of objects sangat umum digunakan untuk data dari database atau API (seperti daftar user, produk, postingan, dll).

## ğŸ› ï¸ Method-Method Penting pada Object

<a name="object-keys"></a>
### ğŸ”‘ Object.keys() - Mendapatkan Semua Key

Method ini mengembalikan **array** berisi semua **key** (nama properti) dari sebuah object.

```javascript
const todo = {
  id: 1,
  name: 'Buy Milk',
  completed: false
};

x = Object.keys(todo);
console.log(x);
// Output: ['id', 'name', 'completed']
```

**Penjelasan:**
- `Object.keys(todo)` mengambil semua key dari object `todo`
- Hasilnya adalah array berisi string
- Urutan key sesuai dengan urutan saat didefinisikan

---

<a name="object-length"></a>
### ğŸ“ Mendapatkan Panjang Object

Object tidak memiliki properti `.length` seperti array. Untuk mendapatkan jumlah properti, gunakan kombinasi `Object.keys()` dan `.length`.

```javascript

// âœ… Cara yang BENAR

console.log(x); // Output: 3
```

**Penjelasan:**
- `Object.keys(todo)` menghasilkan array: `['id', 'name', 'completed']`
- `.length` menghitung jumlah elemen dalam array tersebut
- Hasilnya adalah jumlah properti dalam object (dalam contoh ini: 3)

<a name="object-entries"></a>
### ğŸ­ Object.entries() - Mendapatkan Key-Value Pairs

Method ini mengembalikan **array** berisi **array** yang merepresentasikan pasangan key-value.

- `Object.entries(todo)` mengembalikan array of arrays
- Setiap array dalam memiliki 2 elemen: `[key, value]`
- Berguna saat ingin melakukan iterasi dengan key dan value sekaligus

<a name="has-own-property"></a>
### âœ… hasOwnProperty() - Mengecek Properti

Method ini mengecek apakah object memiliki properti tertentu dan mengembalikan `true` atau `false`.


- Return `true` jika properti ada, `false` jika tidak
- Berguna untuk validasi sebelum mengakses properti
