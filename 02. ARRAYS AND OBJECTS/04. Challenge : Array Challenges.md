

ğŸ“Œ `result` adalah angka `6`, bukan array. Makanya tidak bisa dipanggil `.unshift()`!

**âœ… Yang Benar:**

<a name="kesimpulan"></a>
## ğŸ“ Kesimpulan

### âœ… Apa yang Sudah Dipelajari

1. **Mutating Methods** (`reverse`, `push`, `unshift`, `splice`)
   - Mengubah array asli
   - Tidak bisa di-chain karena return value bukan array

2. **Non-Mutating Methods** (`slice`, `concat`)
   - Membuat array baru
   - Array asli tetap utuh

3. **Spread Operator** (`...`)
   - Cara modern untuk menggabungkan array
   - Lebih readable dan fleksibel

4. **Multiple Solutions**
   - Dalam programming, sering ada banyak cara mencapai hasil yang sama
   - Pilih yang paling mudah dipahami dan sesuai kebutuhan

## ğŸ“– Referensi Method

### `reverse()`

- âœ… Mutating
- Return: array yang sudah dibalik

### `push(item)`
```javascript

```
- âœ… Mutating
- Return: panjang array baru

### `unshift(item)`
```javascript

arr.unshift(1);
console.log(arr); // [1, 2, 3]
```
- âœ… Mutating
- Return: panjang array baru

### `slice(start, end)`
```javascript
// Memotong bagian array (copy)
const arr = [1, 2, 3, 4, 5];
const sliced = arr.slice(1, 4);
console.log(sliced); // [2, 3, 4]
console.log(arr);    // [1, 2, 3, 4, 5] (tidak berubah)
```
- âŒ Not mutating
- Return: array baru hasil potongan

### `concat(array)`
```javascript
// Menggabungkan array

const arr2 = [3, 4];
const combined = arr1.concat(arr2);
console.log(combined); // [1, 2, 3, 4]
```
- âŒ Not mutating
- Return: array baru hasil gabungan

### `splice(start, deleteCount)`
```javascript
// Menghapus/menambah elemen
const arr = [1, 2, 3, 4, 5];
arr.splice(2, 1); // Hapus 1 elemen dari index 2
console.log(arr); // [1, 2, 4, 5]
```
- âœ… Mutating
- Return: array berisi elemen yang dihapus

### Spread Operator `...`
```javascript
// Menyebar elemen array
const arr1 = [1, 2];
const arr2 = [3, 4];
const combined = [...arr1, ...arr2];
console.log(combined); // [1, 2, 3, 4]
```
- âŒ Not mutating
- Tidak ada return (syntax, bukan method)
