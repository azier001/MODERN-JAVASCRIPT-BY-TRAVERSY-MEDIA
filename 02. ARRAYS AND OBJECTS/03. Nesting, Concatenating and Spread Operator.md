## ğŸª† Nesting Arrays (Array Bersarang)

### Apa itu Nesting Arrays?

**Nesting arrays** adalah teknik menempatkan array di dalam array lain. Bayangkan seperti kotak di dalam kotak! ğŸ“¦

<a name="membuat-array-bersarang"></a>
### ğŸ“¥ Membuat Array Bersarang

#### **Cara 1: Menggunakan Method `push()`**

```javascript
// Nesting berries inside of fruits
fruits.push(berries);

console.log(fruits);
// Output: ['apple', 'pear', 'orange', ['strawberry', 'blueberry', 'raspberry']]
```

**Penjelasan:**
- Method `push()` menambahkan `berries` (yang merupakan array) ke dalam array `fruits`
- Sekarang `fruits` memiliki 4 elemen
- Elemen ke-4 adalah array `berries` itu sendiri

#### **Cara 2: Membuat Array Baru dengan Nesting**

```javascript
// Create a new variable and nest both arrays
const allFruits = [fruits, berries];

console.log(allFruits);
// Output: [
//   ['apple', 'pear', 'orange'],
//   ['strawberry', 'blueberry', 'raspberry']
// ]
```

**Penjelasan:**
- Kita membuat array baru `allFruits` yang berisi 2 elemen
- Elemen pertama adalah array `fruits`
- Elemen kedua adalah array `berries`

<a name="mengakses-array-bersarang"></a>
### ğŸ¯ Mengakses Array Bersarang

Untuk mengakses elemen dalam array bersarang, kita menggunakan **double bracket** `[][]`

#### **Contoh 1: Mengakses 'blueberry'**

```javascript
// Now we can access 'blueberry' with the following
x = fruits[3][1];

console.log(x);
// Output: 'blueberry'
```

**Penjelasan:**
- `fruits[3]` â†’ Mengakses elemen index ke-3 (array berries)
- `[1]` â†’ Mengakses elemen index ke-1 dalam array berries ('blueberry')
- Index dimulai dari 0, jadi:
  - 0 = 'apple'
  - 1 = 'pear'
  - 2 = 'orange'
  - 3 = ['strawberry', 'blueberry', 'raspberry']

#### **Contoh 2: Mengakses 'raspberry'**

```javascript
x = allFruits[1][2];

console.log(x);
// Output: 'raspberry'
```

**Penjelasan:**
- `allFruits[1]` â†’ Mengakses array kedua (berries)
- `[2]` â†’ Mengakses elemen index ke-2 ('raspberry')

> ğŸ’¡ **Tips:** Pikirkan bracket pertama `[1]` sebagai "kotak luar" dan bracket kedua `[2]` sebagai "item di dalam kotak"

## ğŸ”— Concatenating Arrays (Menggabungkan Array)

**Concatenating** adalah proses menggabungkan dua atau lebih array menjadi satu array, **BUKAN** membuat array bersarang. Semua elemen akan berada di level yang sama.

<a name="method-concat"></a>
### ğŸ”§ Method `concat()`

```javascript
// concat() - Concatenate arrays
x = fruits.concat(berries);

console.log(x);
// Output: ['apple', 'pear', 'orange', 'strawberry', 'blueberry', 'raspberry']
```

**Penjelasan:**
- Method `concat()` menggabungkan array `berries` ke dalam `fruits`
- Hasilnya adalah **satu array** dengan 6 elemen
- Semua elemen berada di level yang sama (tidak ada array di dalam array)

**Perbedaan dengan `push()`:**
- `push()` â†’ Menambahkan array sebagai satu elemen (nesting)
- `concat()` â†’ Menggabungkan semua elemen dari kedua array

<a name="spread-operator"></a>
### âœ¨ Spread Operator (`...`)

Spread operator adalah cara modern dan lebih fleksibel untuk menggabungkan array!

```javascript
// spread operator (...) - Concatenate with
x = [...fruits, ...berries];

console.log(x);
// Output: ['apple', 'pear', 'orange', 'strawberry', 'blueberry', 'raspberry']
```

**Penjelasan:**
- `...` (tiga titik) adalah **spread operator**
- `...fruits` â†’ "Sebarkan" semua elemen dari fruits
- `...berries` â†’ "Sebarkan" semua elemen dari berries
- Hasilnya sama seperti `concat()`, tapi lebih modern dan fleksibel!

#### **Keuntungan Spread Operator:**

âœ… Lebih mudah dibaca
âœ… Bisa menambahkan elemen tambahan di tengah
âœ… Bisa menggabungkan lebih dari 2 array sekaligus

**Contoh Fleksibilitas:**

```javascript
// Menambahkan elemen tambahan
x = [...fruits, 'mango', ...berries, 'grape'];
// Output: ['apple', 'pear', 'orange', 'mango', 'strawberry', 'blueberry', 'raspberry', 'grape']

// Menggabungkan 3 array
const vegetables = ['carrot', 'potato'];
x = [...fruits, ...berries, ...vegetables];
```

> ğŸŒŸ **Best Practice:** Gunakan spread operator untuk code yang lebih modern dan mudah dibaca!

<a name="flattening-arrays"></a>
## ğŸšï¸ Flattening Arrays (Meratakan Array)

**Flattening** adalah proses mengubah array bersarang menjadi array tunggal (meratakan level-nya).

```javascript
// flat() - Flatten an array
const arr = [1, 2, [3, 4, 5], 6, [7, 8]];
x = arr.flat();

console.log(x);
// Output: [1, 2, 3, 4, 5, 6, 7, 8]
```

**Penjelasan:**
- Array `arr` memiliki 2 array bersarang: `[3, 4, 5]` dan `[7, 8]`
- Method `flat()` "meratakan" semua array bersarang
- Hasilnya adalah satu array dengan semua angka di level yang sama

#### **Visualisasi:**

**Sebelum flat():**
```
[1, 2, [3, 4, 5], 6, [7, 8]]
      â†‘          â†‘
   nested     nested
```

**Setelah flat():**
```
[1, 2, 3, 4, 5, 6, 7, 8]
   Semua di level yang sama!
```

> ğŸ’¡ **Note:** Secara default, `flat()` hanya meratakan 1 level. Untuk level lebih dalam, gunakan `flat(2)`, `flat(3)`, dst.

<a name="static-methods"></a>
## ğŸ”¨ Static Methods pada Array Object

Static methods adalah method yang dipanggil langsung dari **Array Object** (dengan huruf A kapital), bukan dari instance array.

<a name="isarray"></a>
### âœ… `Array.isArray()`

Method ini digunakan untuk **mengecek apakah sesuatu adalah array atau bukan**.

```javascript
// isArray() - Check if is an array
x = Array.isArray(fruits);

console.log(x);
// Output: true
```

**Penjelasan:**
- `Array.isArray(fruits)` mengecek apakah `fruits` adalah array
- Hasilnya `true` karena `fruits` memang array

**Contoh Lain:**

```javascript
Array.isArray([1, 2, 3]);     // true
Array.isArray('hello');        // false
Array.isArray(123);            // false
Array.isArray({name: 'John'}); // false
```

> ğŸ¯ **Kapan digunakan?** Berguna saat bekerja dengan DOM atau data yang tidak pasti tipe datanya.

<a name="from"></a>
### ğŸ”„ `Array.from()`

Method ini **mengubah array-like object atau string menjadi array**.

```javascript
// from() - Create an array from an array-like value
x = Array.from('12345');

console.log(x);
// Output: ['1', '2', '3', '4', '5']
```

**Penjelasan:**
- `Array.from('12345')` mengubah string menjadi array
- Setiap karakter menjadi elemen array tersendiri

**Contoh Lain:**

```javascript

// Output: ['h', 'e', 'l', 'l', 'o']

Array.from({length: 3}, (_, i) => i);
// Output: [0, 1, 2]
```

> ğŸ’¡ **Use Case:** Sangat berguna untuk mengubah NodeList atau HTMLCollection menjadi array!

<a name="of"></a>
### ğŸ“¦ `Array.of()`

Method ini **membuat array dari sekumpulan nilai atau variabel**.

```javascript
// of() - Create an array from a set of values
const a = 1;
const b = 2;
const c = 3;

x = Array.of(a, b, c);

console.log(x);
// Output: [1, 2, 3]
```

**Penjelasan:**
- `Array.of(a, b, c)` membuat array dari nilai variabel a, b, dan c
- Hasilnya adalah array `[1, 2, 3]`

**Contoh Lain:**

```javascript
Array.of(7);        // [7]
Array.of(1, 2, 3);  // [1, 2, 3]
Array.of('a', 'b'); // ['a', 'b']
```

**Perbedaan dengan Array constructor:**

```javascript
Array(7);      // [empty Ã— 7] - Membuat array kosong dengan 7 slot
Array.of(7);   // [7] - Membuat array dengan satu elemen: 7
```
