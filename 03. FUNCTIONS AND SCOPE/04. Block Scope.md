<a name="var-window-object"></a>
## ğŸªŸ var dan Window Object

Ada satu perbedaan lagi yang penting:

```javascript

```

**Penjelasan:**
- Ketika kita membuat variabel global dengan `var`, variabel tersebut **ditambahkan ke window object**
- Ketika kita membuat variabel global dengan `let` atau `const`, variabel tersebut **TIDAK ditambahkan ke window object**

**Mengapa ini penting?**
- Window object adalah object global di browser
- Menambahkan banyak variabel ke window object bisa menyebabkan konflik
- Lebih baik gunakan `let` atau `const` untuk menghindari masalah ini

> âš ï¸ **Praktik Buruk**: Menggunakan `var` untuk variabel global akan menambahkannya ke window object
