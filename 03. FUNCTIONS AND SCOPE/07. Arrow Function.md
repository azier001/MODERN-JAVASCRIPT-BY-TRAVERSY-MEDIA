Arrow function **sangat berguna** ketika digunakan sebagai **callback function** (function yang dilempar ke function lain).

### ğŸ“– Apa itu Callback?

Callback adalah function yang **dioper sebagai parameter** ke function lain.

- `forEach()` adalah method yang **loop** semua elemen array
- Parameter `n` = setiap angka dalam array
- Arrow function membuat kode lebih **clean** dan **mudah dibaca**

**ğŸ¯ Keuntungan:**
