## ğŸ”„ Apa itu Nested Functions?

**Nested Functions** adalah fungsi yang ditulis di dalam fungsi lain. Fungsi di dalam disebut **child function** (fungsi anak), sedangkan fungsi di luar disebut **parent function** (fungsi induk).

```javascript
function parent() {
  // Ini parent function
  
  function child() {
    // Ini child function (nested)
  }
}
```
