<a name="apa-itu-min-dan-max"></a>
### ğŸ¤” Apa itu Min dan Max?

- **Min (Minimum)** = Nilai **terkecil** dalam sekumpulan angka
- **Max (Maximum)** = Nilai **terbesar** dalam sekumpulan angka

**Contoh:**

- Min = **3** (angka terkecil)
- Max = **20** (angka terbesar)


**â— Masalah:** `Math.min()` dan `Math.max()` **TIDAK** menerima array, tapi menerima **angka terpisah**!


<a name="apa-itu-spread-operator"></a>
### ğŸŒŸ Apa itu Spread Operator?

**Spread operator** (`...`) digunakan untuk **"membuka"** isi array menjadi elemen-elemen terpisah.

**Contoh:**
```javascript
const numbers = [1, 2, 3, 4, 5];

// Tanpa spread operator
console.log(numbers); // Output: [1, 2, 3, 4, 5] (masih array)

// Dengan spread operator
console.log(...numbers); // Output: 1 2 3 4 5 (elemen terpisah)
```

**Aplikasi ke Math.min():**
```javascript
const numbers = [1, 2, 3, 4, 5];

Math.min(...numbers); // Sama seperti: Math.min(1, 2, 3, 4, 5)
// Output: 1
```

<a name="apa-itu-iife"></a>
### ğŸ¤” Apa itu IIFE?

**IIFE** = **I**mmediately **I**nvoked **F**unction **E**xpression

Artinya: Function yang **langsung dijalankan** begitu didefinisikan (tanpa perlu dipanggil lagi).

**Struktur IIFE:**
```javascript
// Format dasar IIFE
(function() {
  console.log("Saya langsung jalan!");
})();
```

**Dengan Arrow Function:**
```javascript
(() => {
  console.log("Saya juga langsung jalan!");
})();
```

**Penjelasan:**
1. ğŸ“Œ `(parameter1, parameter2) => { ... }` â†’ Definisi function
