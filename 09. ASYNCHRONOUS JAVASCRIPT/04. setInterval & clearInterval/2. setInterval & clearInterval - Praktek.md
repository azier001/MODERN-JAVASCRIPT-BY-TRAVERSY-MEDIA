<a name="apa-itu-setinterval"></a>
## ğŸ“– Apa itu setInterval?

**setInterval** adalah fungsi JavaScript yang akan **memanggil fungsi secara berulang-ulang** dengan jeda waktu yang tetap.

**Sintaks:**
```javascript
setInterval(fungsi, waktuDalamMilidetik);
```

**Contoh:**
```javascript
setInterval(myFunction, 1000); // Memanggil myFunction setiap 1 detik
```

**Catatan Penting:** â°
- Waktu dihitung dalam **milidetik** (1 detik = 1000 milidetik)
- setInterval akan terus berjalan sampai dihentikan

---

<a name="apa-itu-clearinterval"></a>
## ğŸ›‘ Apa itu clearInterval?

**clearInterval** adalah fungsi untuk **menghentikan** interval yang sedang berjalan.

**Sintaks:**
```javascript
clearInterval(intervalID);
```

**Penjelasan:**
- Kita perlu menyimpan ID dari setInterval
- ID ini digunakan untuk menghentikan interval tersebut

Mari kita pecah **step by step**:

**Step 1: Math.random()**

**Step 3: Math.floor()**

- Buang angka desimal
- Dapat angka bulat

**Step 4: .toString(16)**
```javascript
(2071067).toString(16) // Menghasilkan: "1f99ab"
```
- Ubah angka desimal jadi **hexadecimal**
- Angka 16 = sistem hexadecimal (0-9, A-F)

**Hasil Akhir:**
- Variable `randomColor` isi: `"1f99ab"`
- Ini adalah kode warna hex!
