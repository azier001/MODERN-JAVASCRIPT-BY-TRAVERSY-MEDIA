<a name="apa-itu-setinterval"></a>
## ğŸ“– Apa itu setInterval?

**setInterval** adalah fungsi JavaScript yang akan **menjalankan fungsi atau kode secara berulang-ulang** dengan jeda waktu yang tetap di antara setiap pemanggilan.

### ğŸ”‘ Sintaks Dasar:

```javascript
setInterval(function, delay, arg1, arg2, ...);
```

**Parameter:**
- `function` - Fungsi yang akan dijalankan berulang kali
- `delay` - Waktu jeda dalam **milidetik** (1000ms = 1 detik)
- `arg1, arg2, ...` - Argumen opsional yang akan diteruskan ke fungsi

**Return Value:**
- Mengembalikan sebuah **interval ID** (angka unik) yang bisa digunakan untuk menghentikan interval

<a name="apa-itu-clearinterval"></a>
## ğŸ›‘ Apa itu clearInterval?

**clearInterval** adalah fungsi yang digunakan untuk **menghentikan interval** yang sedang berjalan.

### ğŸ”‘ Sintaks:

```javascript

```

**Parameter:**
- `intervalID` - ID yang dikembalikan oleh `setInterval()`

> âš ï¸ **Penting:** Tanpa `clearInterval()`, interval akan terus berjalan selamanya!

## âš™ï¸ Cara Kerja setInterval

<a name="fungsi-random-color"></a>
### ğŸŒˆ Fungsi Random Color

Ini adalah fungsi yang **menghasilkan warna acak** dalam format hexadecimal:

```javascript

  const randomColor = Math.floor(Math.random() * 16777215).toString(16);
  document.body.style.backgroundColor = `#${randomColor}`;
}
```

### ğŸ” Penjelasan Detail:

**1. Menghasilkan Angka Random:**
```javascript
const randomColor = Math.floor(Math.random() * 16777215).toString(16);
```

Mari kita pecah menjadi bagian-bagian kecil:

| Bagian | Penjelasan | Contoh |
|--------|------------|--------|
| `Math.random()` | Menghasilkan angka desimal random 0 - 0.999... | `0.745` |
| `* 16777215` | Dikali 16777215 (jumlah total warna hex) | `12489134.325` |
| `Math.floor()` | Membulatkan ke bawah (hilangkan desimal) | `12489134` |
| `.toString(16)` | Konversi angka ke hexadecimal (basis 16) | `"be68ce"` |

> ğŸ’¡ **Kenapa 16777215?**
> - Warna hex format: `#RRGGBB` (6 digit)
> - Setiap digit bisa 0-F (16 kemungkinan)
> - Total kombinasi: 16^6 = 16,777,216 warna (0 sampai 16777215)

**2. Set Background Color:**
```javascript
document.body.style.backgroundColor = `#${randomColor}`;
```

**Penjelasan:**
- Template literal `` `#${randomColor}` `` menggabungkan `#` dengan nilai `randomColor`
- Contoh: jika `randomColor` = `"be68ce"`, hasilnya: `"#be68ce"`
- `document.body.style.backgroundColor` diset dengan warna hex tersebut

**Contoh Hasil:**
```javascript

// Background jadi: #a3f5b2 (hijau muda)
```
