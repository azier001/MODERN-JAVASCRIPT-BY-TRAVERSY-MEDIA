<a name="5-handler-onreadystatechange"></a>
### 5ï¸âƒ£ Handler onreadystatechange

```javascript
  xhr.onreadystatechange = function () {
    if (this.readyState === 4) {
      if (this.status === 200) {
        jokeEl.innerHTML = JSON.parse(this.responseText).value;
      } else {
        jokeEl.innerHTML = 'Something Went Wrong (Not Funny)';
      }
    }
  };
```

**Penjelasan Detail:**

#### ğŸ”¢ `readyState` - Status Request

| Value | Arti |
|-------|------|
| 0 | Request belum diinisialisasi |
| 1 | Server connection established |
| 2 | Request received |
| 3 | Processing request |
| **4** | **Request finished & response ready** âœ… |

> âš¡ Kita check `readyState === 4` karena ini artinya response sudah siap!

#### ğŸ“Š `status` - HTTP Status Code

| Code | Arti |
|------|------|
| **200** | **OK - Success** âœ… |
| 404 | Not Found âŒ |
| 500 | Server Error âŒ |

> âœ… Status `200` = Request berhasil!

#### ğŸ”„ Nested If Statement

```javascript
if (this.readyState === 4) {           // Check: response ready?
  if (this.status === 200) {           // Check: request successful?
    // Tampilkan joke
  } else {
    // Tampilkan error
  }
}
```

**Kenapa nested?**
- If pertama: Pastikan response sudah ready
- If kedua: Pastikan request berhasil (status 200)
- Else: Handle error jika status bukan 200

<a name="6-parsing-json-response"></a>
### 6ï¸âƒ£ Parsing JSON Response

```javascript
jokeEl.innerHTML = JSON.parse(this.responseText).value;
```

**Mari kita breakdown:**

#### ğŸ”¤ `this.responseText`
- Berisi response dari server dalam bentuk **string**
- Formatnya seperti object, tapi masih string!

**Contoh:**
```javascript
// this.responseText (masih string):
'{"value":"Chuck Norris joke here","id":"abc123"}'
```

#### ğŸ”§ `JSON.parse()`
- Mengubah string JSON menjadi **object JavaScript**
- Setelah di-parse, kita bisa akses propertynya

**Setelah parse:**
```javascript
// Jadi object JavaScript:
{
  value: "Chuck Norris joke here",
  id: "abc123"
}
```

#### ğŸ“¦ `.value`
- Mengambil property `value` dari object
- Ini adalah joke yang kita mau!

**Step by step:**
```javascript
// 1. Response text (string)
this.responseText
// "{"value":"Chuck Norris can divide by zero",...}"

// 2. Parse jadi object
JSON.parse(this.responseText)
// {value: "Chuck Norris can divide by zero", ...}

// 3. Ambil property value
JSON.parse(this.responseText).value
// "Chuck Norris can divide by zero"
```

> ğŸ¤” **Kenapa property namanya "value"?**  
> Itu keputusan pembuat API. Memang agak aneh - seharusnya `joke` atau `text` lebih jelas!
