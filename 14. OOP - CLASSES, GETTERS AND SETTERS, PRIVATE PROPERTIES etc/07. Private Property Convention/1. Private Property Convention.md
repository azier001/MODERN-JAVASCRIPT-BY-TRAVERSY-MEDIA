<a name="apa-itu-encapsulation"></a>
## 🔍 Apa itu Encapsulation?

Encapsulation itu intinya adalah **membungkus data dan method ke dalam satu class**, sekaligus **menyembunyikan** bagian-bagian yang tidak perlu diakses dari luar.

Di bahasa seperti Java, ada keyword `private` yang benar-benar memblokir akses dari luar class. Tapi di JavaScript? Tidak ada keyword seperti itu (setidaknya sampai ES2022). Makanya komunitas JavaScript punya **konvensi** sendiri.

> 💡 **ES2022** memperkenalkan `#` (tanda pagar) untuk membuat *truly private fields*. Tapi untuk sekarang, konvensi underscore masih lebih umum ditemukan di kode-kode yang ada.

---

<a name="private-property-konvensi-underscore"></a>
## 🔒 Private Property: Konvensi Underscore

Caranya simpel: tambahkan `_` di depan nama property atau method yang ingin kamu tandai sebagai *private*.

```js
this._balance = 0;      // ✅ konvensi: ini private, jangan diakses langsung dari luar
this.balance = 0;       // ini public
```

**Penting dipahami:** underscore ini **bukan penghalang teknis**. Kamu masih bisa akses `wallet._balance` dari luar dan itu tetap jalan. Tujuannya cuma satu — **komunikasi ke developer lain** (atau dirimu sendiri di masa depan) bahwa *"hei, jangan sentuh ini dari luar class"*.

<a name="membangun-class-wallet"></a>
## 🏗️ Membangun Class Wallet

Kita akan pakai contoh **dompet digital** — mirip aplikasi bank atau crypto wallet. Class ini punya:

- `_balance` → saldo (private)
- `_transactions` → riwayat transaksi (private)

```js
class Wallet {
  constructor() {
    this._balance = 0;
    this._transactions = [];
  }
}
```

Constructor tidak menerima parameter apapun. Semua diinisialisasi dari dalam.

---

<a name="method-deposit--withdraw"></a>
## 💰 Method Deposit & Withdraw

Ini adalah **public method** — pintu resmi untuk berinteraksi dengan wallet dari luar class.

```js
deposit(amount) {
  this._processDeposit(amount);   // panggil private method dulu
  this._balance += amount;        // baru ubah saldo
}

withdraw(amount) {
  if (amount > this._balance) {
    console.log('Not enough funds');
    return;                        // stop eksekusi jika saldo kurang
  }

  this._processWithdraw(amount);  // panggil private method
  this._balance -= amount;        // kurangi saldo
}
```

Ada validasi di `withdraw`: kalau jumlah yang mau ditarik **lebih besar dari saldo**, transaksi dibatalkan dan muncul pesan `"Not enough funds"`.

<a name="getter-cara-aman-akses-data-private"></a>
## 👁️ Getter: Cara Aman Akses Data Private

Karena `_balance` private, kita butuh cara untuk membaca nilainya tanpa mengakses property itu langsung. Solusinya: **getter**.

```js
get balance() {
  return this._balance;
}

get transactions() {
  return this._transactions;
}
```

Cara pakainya persis seperti property biasa (tanpa tanda kurung):

```js
console.log(wallet.balance);       // ✅ pakai getter
console.log(wallet.transactions);  // ✅ pakai getter

// ❌ hindari ini (meskipun masih bisa jalan):
console.log(wallet._balance);
console.log(wallet._transactions);
```

> 💡 Keuntungan pakai getter: kalau suatu saat kamu mau ubah format atau tambahkan logika sebelum return, kamu bisa lakukannya di dalam getter tanpa ubah cara penggunaan di luar.

---

<a name="menyimpan-riwayat-transaksi"></a>
## 📋 Menyimpan Riwayat Transaksi

Setiap deposit dan withdraw akan merekam aktivitasnya ke array `_transactions`. Setiap entri berupa object dengan dua key:

```js
{
  type: 'deposit',   // atau 'withdraw'
  amount: 300
}
```

Untuk lihat riwayat:

```js
console.log(wallet.transactions);
// Output:
// [
//   { type: 'deposit', amount: 300 },
//   { type: 'withdraw', amount: 50 }
// ]
```

<a name="private-method-processdeposit--processwithdraw"></a>
## 🔧 Private Method: processDeposit & processWithdraw

Ini method internal yang dipanggil **dari dalam class**, bukan dari luar. Makanya diberi prefix underscore.

```js
_processDeposit(amount) {
  console.log(`Depositing ${amount}`);

  this._transactions.push({
    type: 'deposit',
    amount,
  });
}

_processWithdraw(amount) {
  console.log(`Withdrawing ${amount}`);

  this._transactions.push({
    type: 'withdraw',
    amount,
  });
}
```

Tugasnya:
1. Log aktivitas ke console
2. Catat transaksi ke array `_transactions`

Pengguna wallet tidak perlu tahu method ini ada. Mereka cukup panggil `deposit()` atau `withdraw()`, sisanya diurus otomatis di balik layar.

<a name="analogi-mobil--encapsulation"></a>
## 🚗 Analogi: Mobil & Encapsulation

Ini analogi yang bagus buat ngerti kenapa encapsulation penting:

| Yang kamu tahu (public) | Yang tersembunyi (private) |
|---|---|
| 🦶 Gas pedal → `deposit()` | ⚙️ Alternator → `_processDeposit()` |
| 🛑 Rem → `withdraw()` | 🔩 Axle roda → `_processWithdraw()` |
| 🔑 Kunci kontak → `balance` getter | 🔋 Sistem pengapian → `_balance` |

Kamu bisa nyetir mobil tanpa harus ngerti cara kerja alternator. Sama persis: pengguna class `Wallet` bisa deposit/withdraw tanpa harus tahu gimana `_processDeposit` bekerja di balik layar.

<a name="kode-lengkap"></a>
## 📝 Kode Lengkap

```js
class Wallet {
  constructor() {
    this._balance = 0;
    this._transactions = [];
  }

  // ✅ Public method
  deposit(amount) {
    this._processDeposit(amount);
    this._balance += amount;
  }

  // ✅ Public method
  withdraw(amount) {
    if (amount > this._balance) {
      console.log('Not enough funds');
      return;
    }

    this._processWithdraw(amount);
    this._balance -= amount;
  }

  // 🔒 Private method
  _processDeposit(amount) {
    console.log(`Depositing ${amount}`);

    this._transactions.push({
      type: 'deposit',
      amount,
    });
  }

  // 🔒 Private method
  _processWithdraw(amount) {
    console.log(`Withdrawing ${amount}`);

    this._transactions.push({
      type: 'withdraw',
      amount,
    });
  }

  // 👁️ Getter untuk akses _balance
  get balance() {
    return this._balance;
  }

  // 👁️ Getter untuk akses _transactions
  get transactions() {
    return this._transactions;
  }
}

// --- Penggunaan ---
const wallet = new Wallet();

wallet.deposit(300);     // Depositing 300
wallet.withdraw(50);     // Withdrawing 50

console.log(wallet.balance);       // 250
console.log(wallet.transactions);  // [{type:'deposit',amount:300},{type:'withdraw',amount:50}]
```
