<a name="constructor-function"></a>
## ğŸ—ï¸ Constructor Function

Kita akan membuat `Person` dengan dua properti: `firstName` dan `lastName`. Tapi kita simpan nilainya di properti "tersembunyi" dengan awalan underscore (`_`), supaya getter-nya bisa punya nama yang bersih.

```javascript
function Person(firstName, lastName) {
  this._firstName = firstName; // properti asli, disimpan dengan underscore
  this._lastName = lastName;
}
```

> ğŸ’¡ Kenapa pakai underscore? Ini konvensi umum untuk menandai bahwa properti ini "private" â€” jangan diakses langsung dari luar, gunakan getter-nya.

<a name="menambahkan-getter--setter"></a>
## âœ¨ Menambahkan Getter & Setter

Untuk menambahkan getter dan setter ke constructor function, kita pakai **`Object.defineProperty()`**. Cara ini sedikit berbeda dari class, tapi hasilnya sama.

`Object.defineProperty()` menerima **3 argumen**:
1. **Objeknya** â†’ `this`
2. **Nama properti** yang ingin kita beri getter/setter â†’ misalnya `'firstName'`
3. **Objek konfigurasi** yang berisi fungsi `get` dan `set`

```javascript
function Person(firstName, lastName) {
  this._firstName = firstName;
  this._lastName = lastName;

  // Getter & Setter untuk firstName
  Object.defineProperty(this, 'firstName', {
    get: function () {
      return this._firstName; // mengembalikan nilai _firstName
    },
    set: function (value) {
      this._firstName = value; // mengubah nilai _firstName
    },
  });

  // Getter & Setter untuk lastName
  Object.defineProperty(this, 'lastName', {
    get: function () {
      return this._lastName;
    },
    set: function (value) {
      this._lastName = value;
    },
  });
}
```

Sekarang kita bisa buat objek dan akses propertinya:

```javascript
const person1 = new Person('john', 'doe');

console.log(person1.firstName); // "john"
console.log(person1.lastName);  // "doe"
```

> ğŸ”‘ Perhatikan: kita akses `person1.firstName` (tanpa underscore), bukan `person1._firstName`. Karena kita ingin lewat getter, bukan langsung ke properti aslinya.

## ğŸ”¡ Helper: `capitalizeFirst`

Sekarang kita tambahkan fitur biar nama yang diinput huruf kecil bisa otomatis **kapitalisasi huruf pertamanya**. Caranya dengan menambahkan method `capitalizeFirst` ke `prototype` dari `Person`.

```javascript

  return value.charAt(0).toUpperCase() + value.slice(1);
  //     ^ambil huruf pertama    ^huruf besar  ^gabung dengan sisa string
};
```

```javascript

    return this.capitalizeFirst(this._firstName); // âœ… pakai helper
  },

    this._firstName = value;
  },

```

Sekarang hasilnya:


<a name="getter-fullname"></a>
## ğŸ”— Getter `fullName`

Yang keren: kita bisa **pakai getter lain di dalam getter**! Jadi getter `fullName` bisa memanggil getter `firstName` dan `lastName` yang sudah punya kapitalisasi.

```javascript

  get: function () {
    // this.firstName â†’ memanggil getter, bukan _firstName langsung
    // jadi sudah otomatis terkapitalisasi
    return this.firstName + ' ' + this.lastName;
  },
  // tidak perlu setter untuk fullName
});
```

```javascript

```

> ğŸ’¡ Karena `this.firstName` (tanpa underscore) akan otomatis memanggil getter-nya, kita tidak perlu memanggil `capitalizeFirst` lagi secara manual di sini.

**Kode lengkap Constructor Function:**

```javascript

  this._firstName = firstName;
  this._lastName = lastName;

  Object.defineProperty(this, 'firstName', {
    get: function () {
      return this.capitalizeFirst(this._firstName);
    },
    set: function (value) {
      this._firstName = value;
    },
  });

  Object.defineProperty(this, 'lastName', {
    get: function () {
      return this.capitalizeFirst(this._lastName);
    },
    set: function (value) {
      this._lastName = value;
    },
  });

  Object.defineProperty(this, 'fullName', {
    get: function () {
      return this.firstName + ' ' + this.lastName;
    },
  });
}

Person.prototype.capitalizeFirst = function (value) {
  return value.charAt(0).toUpperCase() + value.slice(1);
};

const person1 = new Person('john', 'doe');
console.log(person1.firstName); // "John"
console.log(person1.lastName);  // "Doe"
console.log(person1.fullName);  // "John Doe"
```

<a name="object-literal"></a>
## ğŸ“¦ Object Literal

Cara kedua adalah dengan **object literal**. Bedanya dengan constructor function, object literal itu langsung satu objek jadi â€” bukan blueprint untuk membuat banyak objek.

Cara nulis getter dan setter di object literal **mirip dengan class** â€” tidak perlu `Object.defineProperty()`. Cukup pakai kata kunci `get` dan `set` langsung di dalam objek.

```javascript
const PersonObj = {
  _firstName: 'jane', // properti dengan underscore
  _lastName: 'doe',

  // Getter firstName
  get firstName() {
    return Person.prototype.capitalizeFirst(this._firstName);
  },

  // Setter firstName
  set firstName(value) {
    this._firstName = value;
  },

  // Getter lastName
  get lastName() {
    return Person.prototype.capitalizeFirst(this._lastName);
  },

  // Setter lastName
  set lastName(value) {
    this._lastName = value;
  },

  // Getter fullName (pakai getter lain, tanpa setter)
  get fullName() {
    return this.firstName + ' ' + this.lastName;
  },
};
```

> ğŸ“ Di sini kita "meminjam" fungsi `capitalizeFirst` dari `Person.prototype` supaya tidak nulis ulang logika yang sama.

<a name="objectcreate"></a>
## ğŸŒ¿ `Object.create()`

Karena `PersonObj` itu object literal (bukan class/constructor), cara membuat objek baru darinya adalah pakai **`Object.create()`**. Hasilnya adalah objek baru yang mewarisi semua properti dan getter dari `PersonObj`.

```javascript
const person2 = Object.create(PersonObj);

console.log(person2.firstName); // "Jane"
console.log(person2.lastName);  // "Doe"
console.log(person2.fullName);  // "Jane Doe"
```

> ğŸ’¡ `Object.create(PersonObj)` membuat objek baru dengan `PersonObj` sebagai prototype-nya â€” mirip seperti `new Person()` pada constructor function.
